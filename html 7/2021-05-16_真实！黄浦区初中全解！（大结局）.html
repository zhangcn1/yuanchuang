    今天说黄浦的公办初中最后一篇！首先把重点说一下！黄浦区正在不断变革中！原因有很多，比如集团化办学和学区化办学的深入，又比如动拆迁的影响！黄浦区从去年开始到今年不停的在调整和变化，而且未来几年乐爸可以几乎肯定的说依旧会继续变化！至于有哪些变化呢？乐爸细细数来。

    先看几张对比图：

 

    

    以上是2020年黄浦公办小升初和2021年的对比！

我们可以看到的是小升初对口还是发生了不少变化的！总结如下：

1.     报童小学从对口应昌围棋变成了金陵中学。（原因应昌围棋学校被格致并了）

2.     复兴中路第二小学今年木有了！对口的四个居委复四、复兴、合一、合二，分别给了复四，合二和复兴给了卢湾三中心小学，合一居委不知道分到哪里去了，乐爸不知道，改天去看看！所以卢湾实验小学就唯一对口比乐了！

3.     应昌围棋学校小学部招生并到了北京东路小学！所以曹光彪不在唯一对应格致初级，而是对口多了个北京东路小学合以前的应昌的生源！

4.     光明小学今年招生没了，基本并入了复兴东路第三小学的范围内！对口也变成了光明初级中学。复兴东路之前对口的是黄浦学校！黄浦小学只对口自己的本部小学部了！

大家看到了吗？仅仅2020年和2021年至今就差了那么多！（其实去年也有变化，比如卢湾二中心对口换成了向明！）

    未来黄浦的学校和动迁还有会有，集团化办学进一步在进行，比如坊间的传闻蓬莱路二小的搬迁！比如新学校的建立！对口和居委是否会继续改变这是未知的，除非周边很完善，比如类似黄外这样的学校不会变动外，其他乐爸真的不知道如何变化！

    而这样的变化也千丝万缕的关联着黄浦初中的生源情况！（大家都知道生源的重要性）

    那么今天我们就开始讲目前来说黄浦公办初中的情况吧！




一


全解系列


1.     卢湾中学

对口海华小学生源，20年10个推优，21年11个推优，中考人数157人的样子！

直接说具体情况：

1.     卢湾中学上午七点半到校基本，晚上放学的话要看具体年级和什么班级了！

2.     卢湾中学预初就开始有月考，抓的还是比较紧的。

3.     学校有好班，好班数量在3个的样子，最好的理科班市重点率在90%以上，四校也有，八大+向明一半的样子。（含推优+名额分配）

4.     外招不算太多，主要是海华的生源，校风很不错，但生源还不是顶尖，有这个成绩，学校算很努力了。

5.     学习相对比较低调一点，中考的话成绩基本排在格致和大同后面（不包括民办），和向明相比有的一比，但向明目前更好一点点（几乎可以忽略不计），家长的反馈是卢湾初级抓的比向明紧一点。

6.     学校中生代老师和年轻老师比较多，可以和学生打成一片，还是很不错的！

7.     有个爹，还算不错！

 

 

2.     尚文中学

对口黄浦一中心小学和董家渡路第二小学，21年推优6个，20年7个，大概中考人数在100人的样子！

直接说情况：

1.     对口2个小学的情况下，每个年级人数不足150个人，预初6个班，其他年级4-5个班，所以每个班的人数真的少！乐爸估算有一半的对口的学生去了别的学校，特别是黄浦一中心。

2.     有个好班，以前有分班考，20年没了，好班和平行班差距比较大，平行班前三的话在年级也就排30-40的样子，大家脑补！

3.     没有爹，比较吃亏一点！

4.     校服颜色比较多样化，男生淡蓝色，女生粉色，还有套绿色的秋冬校服！

5.     学校好班有周周爽的，特别是英语方面。

6.     平行班就呵呵哒了，学风不咋地，师资也是有倾斜的，初二历史上有再分班的情况发生。

7.     好班中考还是不错的，有四校分校，有八大，其他市重点的数量也不少，平行班极个别能进市重点不错了。（高中都是个位数）

8.     活动挺多的，什么艺术节之类的各种节，和黄浦一中心小学还有结对子义卖。

9.     每天中午吃饭结束有休息，很多娃喜欢去操场散步，这个和民办简直就没法比了。

10.   初三的时候学校对好班抓的比较紧，但其他的嘛，呵呵哒，有娃和我吐槽，他们初三了，平行班还有斗地主的！

 

结论：普中，能进好班的话，出路还是有的。

 

 

3.     光明初级

对口复兴东路第三小学，20年和21年的推优都是2个，妥妥的人少！中考人数大概在29人的样子！

以前很好的学校，老上海都知道的，但是现在嘛，呵呵哒，整体中下水平了！中考人数就看出来了！

具体情况：

1.     学校地段贼好，边上就是城隍庙，不过边上都拆完了，生源也是呵呵哒，城隍庙的小商贩和老板的娃在里面居多。

2.     学校历史悠久的一塌糊涂。

3.     学校有个爹，但和爹没啥关系，有点法语特色。

4.     乐爸门口蹲点看见3对CP。（确定肯定是CP）

5.     有个比较优秀的娃告诉我他们有几个好老师，乐爸特地记了一下，语文周杰伦老师（我不确定是不是听错了，额），英语有个姓张的老师，这两个老师很好。

总结，：我也不知道咋总结了1

 

 

4.     市八初级中学

对口中华路第三小学，21年推优6个，20年7个，大概中考人数在100人的样子！

直接说具体情况：

1.     有好班，还是2个！额！

2.     以前叫多稼中学，平行班呵呵哒，中考录取率一届也就20个不到的样子！

3.     整体黄浦中等偏下一点点。

4.     接受统筹学生。

5.     市八很早之前还不错的，现在嘛，额！

6.     一个班级20人左右，绝对是小班化，没办法的事情。

7.     校服是深红色。

8.     20年预初4个班。

9.     学生的流失率到不是很大，这点还挺奇怪的。

 

 

5.     兴业中学

对口卢湾一中心，卢湾三中心，20年推优有7个，100个人中考的样子，但21年只有4个推优，58个人中考的样子！

直接说具体情况：

1.     对口的小学其实还不差的，但卢湾一中心对口大部分都打死不去的，各有各的方法。

2.     老师还是比较认真的，兴业牛的娃也基本来自于卢湾一中心。

3.     有好班，和平行班差距巨大无比！

4.     兴业很久以前也不错的，历史上还有一段时间被叫做小向明，但现在生源真的是越来越少，所以只能保一个好班了！和之前提到的光明中学有合作法语特色！

5.     学校的想法是走特色，但好像特色有点多！英语法语，艺术（打击乐，剪纸），科技，体育（射箭，击剑）。。。。。。那么多。。。我想说你们忙得过来嘛？呵呵哒！多特色等于没特色，乐爸吐槽！

总结一下，最多算普中！摇号后，可能卢湾一中心的娃多一点会好起来，这个是唯一值得期待的地方，不过还要看学校的努力了！

 

 

6.     清华中学

对口淮海中路小学和巨鹿路第一小学！20年推优1个，21年推优2个，额！中考人数少于30人！21年才14人左右参加中考！

直接说情况：

1.     黄浦目前来说不多见的菜+菜的配置。（别怪乐爸那么直接）

2.     20年中考名额分配的情况下去向明拿到个名额，结果！没到分数线，额！浪费！

3.     名字太牛，但是真的呵呵哒！

4.     巨鹿路小学瑞金二路街道指定接受统筹的学校，大家脑补一下！

5.     那么点点人中考，这个就类似于某网红新云台了，哈哈哈，黄浦7所市重点！至少7个名额分配，一比二投档，我靠！但是呵呵哒，能到线再说！

6.     学校地段很好，核心未知，教学质量真的呵呵哒，主要是生源！

7.     网球特色学校！

乐爸总结：你能在黄浦找到一个户籍率比他低的我请吃饭，200块以内！

 

 

7.     敬业初级中学

对口实验小学，回民小学，20年和21年的中考推优都是5个，倒推中考人数大概在72人的样子。

直接说情况：

1.     问了一大圈，家长和门卫只知道一个学校特点，“嘉年华”校本课程，乐爸查了半天，叫I-CACM—STEM+课程，也不知道是啥东东！

2.     和敬业关系不大，是以前的翻牌中学，爹的话最多算小半个。

3.     有好班，一个年级4个班的样子。

4.     学生反馈：学校年轻老师和中生代老人很多，占大多数，一个娃说有个数学老师比较有名，姓谷！

乐爸总结：普中一所！



 

8.     黄浦学校

对口自己学校小学部，21年推优3个，20年4个，中考人数在42到58人之间徘徊！

直接说情况：

1.     硬件顶级配置，都快赶上国际学校了，还在市中心！

2.     一流硬件，二流师资，三流生源！

3.     老西门最近拆迁有多少大家也是知道的，所以生源真的呵呵哒了。

4.     体育特色，这个特色还非常奇葩，轮滑！还有个轮滑体育馆！真是阔气！

5.     走体育路线的可以考虑一下，哈哈哈，数学是体育老师教的吗？（乐爸开个玩笑哈）

6.     家长反馈，学校硬件真的多，什么情绪发泄室，舞蹈房，电脑房，厕所也非常气派！

7.     周边10+万一平的房子的家长会让娃去这里吗？额！

8.     老上海都知道，这里以前室沪南体育场，有个阿姨和我说，这个学校么就相当于少体校了呀！（噗，阿姨真懂）

9.     隔壁红房子，正对病房手术室等，哇哈哈哈，有迷信的家长不？乐爸不迷信，坚决觉得无所谓！

总结：中等偏下！

 

 

9.     黄教院附属中山学校

对口中山小学部和瞿溪路小学，21年6个推优，20年5个，中考人数在71到85之间！

直接说情况：

1.     和楼上的黄浦一样的硬件强大！而且区里有什么活动那个也喜欢放这里，但是对口的瞿溪路实在是。。。

2.     让乐爸意外的是，这个学校竟然有好班，还是2个！额！现在普中是不是都喜欢搞好班啊？不过这个好班的质量乐爸就不得而知了。

3.     有个家长反馈，学校其实抓的挺紧的，但吃亏在生源上，自己小学部上来的还算马马虎虎，隔壁小学来的绝对拉低质量了。

4.     学校有不少教室都空着！

5.     据一个阿姨说这几年所谓两个好班单独拿出来能排在黄浦前10！（真的吗？）

6.     干活的副校长被弄到大同初去了！
