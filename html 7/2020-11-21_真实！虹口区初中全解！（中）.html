    虹口所有初中的具体情况！（中）

    虹口也是一个民办相对公办强的多的区！初中四大民办长期占据龙头地位，小学有了民办四中心也逐渐形成了虹口四大民办小学的势头！虹口整体的教育虽然在整个上海没有排很前，但相对少的中考人数以及不错的地理位置，决定了虹口其实也是很不错的一个区！所以！

一


全解系列


1.     曲阳第二中学

招收班级：6

 

今年对口小学为：曲阳三小，曲阳四小，宏星民办，整体来说生源得到较大提升！历年数据来看还算不错，但有起伏，比如有的时候考的很好，有的年份考的一般，学校活动还算丰富，同样高年级后逐渐减少，抓学习！随着宏星的生源加入，学校应该会升级一个档次！

家长反馈：学校目前是迅行教育集团，师资相对比以前好了，学校有2个好班，现在不知道还有没有，好班成绩不错的，平行班么就呵呵哒了！

学生反馈：曲二17年好班2个四大，11个市重点，19个区重点，5个普高，一个中职。

家长反馈：17年是曲二考的最好的一年，当时发红榜的，乐爸查了下，的确是的，但后面几年考的一般般了！

学生反馈 ：吐槽说学校校徽难看，然后学校还是可以的，老师比较负责，巧了，娃在好班！乐爸去看了下，校徽像个大松鼠的尾巴，，哈哈哈哈，其实挺可爱的！


 

2.     虹口实验学校

招收班级：8

 

虹口实验网评一直是虹口第一公办，也是虹口区着力打造的一个公办初中的拳头学校，从学校以前的对口小学一路变到现在的四中心（公+民）和自己的本部学校看来（16年把曲阳四小踢走加了宏星，现在宏星也被踢走，为了保民办四中心），小学生源绝对可以的，历年的中考成绩也是节节攀升，目前是市重点率在30%不到，区重以上的比例在70%上下，可以说在虹口以前民办天下的情况下很可以的数据了，随着民办摇号的产生，未来的成绩应该会更好，超越前三部敢说，和新北郊打一打还是很有底气的！

家长反馈：很巧，也是来问情况的，孩子3岁，说准备买这里，虽然知道小学一般，但好在九年一贯，保底不错，家长是夫妻一起来的，双硕士！

家长反馈：学校有好班，会有拓展，平行班较为一般，不过也有尖子生，高年级考试还是比较频繁，有排位的。

乐爸目测：门口接娃的整体情况，私家车很多，不乏豪车接送的家长，家长整体素质尚可。

学生反馈：小学本部升初中的娃和以前宏星以及四中心的娃还是有很大差别的，特别是开始的成绩上会有明显的分差，不过通过努力可以追回。

学生反馈：虹口实验活动蛮多的，每个班都有什么兴趣课，比如围棋，舞蹈，手风琴啥的，高年级就没了！平行班和好班的学风有差异的！

 

 

3.     五十二中学

招收班级：2

 

网评虹口倒数第一（其实不至于，但也基本差不多），很多人奇怪，为啥对口两个那么好的小学会那么差？因为广灵路小学和丽音的娃是能不去就不去的，导致可能每年只有个位数而且学习不怎么好的会去，其他全部面试和择校了，所以导致学校生源真的不好！

但是今年摇号了！会有一部分没有准备浦东和闵行学区房的家长去摇号，而且大概率都会去摇！没摇中的话那也没办法了！所以估计4年后，52能甩掉菜中的名号！

学校因为有丽音和广灵路的极个别生源，所以中考也会有极少数量的市重点，包括偶尔甚至有四校的！多年总体市重点率绝对不超过10%。有复旦的这一届还贴喜报了，额！

实地探访：

1.     学校早恋的不好，乐爸看到不少！

2.     学风较差，有同学门口放学聊天的时候聊天说到班里的一个学习挺好的同学如何如何，想一起去整整她，额！

3.     学生基本不和乐爸聊，就一个看起来蛮文静的女学生说，学校老师真的很一般！

4.     学校是足球特色，足球很厉害（一个男学生说的）


 

 

4.     北虹初级中学

招收班级：5

 

先来个瓜：陈蓉，马伊俐都是北虹毕业的！对口三个小学，广中路小学，六中心，柳林路小学，学校是百年老校，大概150多年了！

看应试情况来说，在公办里排名中游水平，属于普中级别的。05年在昆山中学的基础上和韶山中学一起合并成了现在的北虹！

这个学校的抽样比较少，乐爸到晚了，说一下情况供参考

1.     学校之前有特色班，但没有那么的特色，到后来平行班也有进控江和复兴的，总体教学难度和深度和民办肯定不能比的，中规中矩！

2.     每年预录取8大的大概个别几个人，有时候2个，有时候4个。

3.     同学反馈：年级前20基本市重点比较稳，另外学校硬件可以的，然后一个同学和乐爸反馈学校的男老师比较多，比女老师多，另外部分男老师比较娘一点（额，这个供参考，听听就好，我觉得娘的男老师其实挺好的）



 

5.     市五中学

招收班级：2

 

对口小学是鼎鼎大名的上外附小，呵呵哒了，同情一下上外附小！以前面试时代基本没人去市五的，有几年甚至是0！为啥？上外附小太牛，除了近50不到的附中（含东校）的录取人数外，其他都基本大概率能面试成功，学校一看上外附小，基本就妥了！但现在摇号了，附小非顶级的娃（没拿到面单）要么放低要求去克勒，要么去浦东闵行买学区房了，去市五？不存在的事情，除非啥都没准备的附小爸妈！

所以市五以前基本都是接受统筹的情况，所以学校情况嘛，呵呵哒一下！但现在因为有变数，未来还真不知道会不会出来几个附小遗珠带动一下市五的成绩！从班级数也可以看出，只有2个班的人！接下来说一下实地采访情况：

1.     学校历年的高中录取率（看清楚，不是区和市重点哈）基本没高于过70%，基本在50%左右徘徊，也就是50%的人连高中都考不上！

2.     学校早恋的蛮多的，反正乐爸看到不少！

3.     随迁子女不少，上海户籍的也有，但乐爸观察下来基本没有交集。

总体学校比较靠后，未来是不是能借摇号起一波乐爸表示比较悲观。



 

6.     江湾初级中学

招收班级：4

 

对口凉城二小，凉城二小也是4个班，有择校和摇号的情况下，初中肯定招不满，所以大家懂了哈！属于虹口比较好的公办初中了，特别在凉城这块！

学校有好班！地段班比较一般！学校硬件可以的，很不错，疫情前也是对外开放操场的，时间是放学后（这点在虹口似乎很普遍），学校前身是完中，后来分离了，高中变民办，初中独立出来，从1999年开始用了大概10年时间从弱慢慢变好。说一下实地考察的内容：

1.     学校的好班抓的蛮紧的，但作业不算多，效率高的学生基本7点前能做完，学习氛围不错，有尖子生，之前几年有进上中和华二的，区内市重点也不少，期末和期中考以及初三作业量会加大。

2.     老师普遍都挺负责的，学生反馈，有几个学生反馈有几个老师挺凶的，很严厉，反馈说各个班的师资还是不同的，一个学生和乐爸反馈很羡慕隔壁好班的师资！（从这点可以看出江湾的学生还是比较上进的）

3.     当然还是有调皮的学生，放学谈论着游戏，比如王者荣耀，约着什么时候爸妈不在家的时候杀个过瘾的话语。

4.     大部分家长和学生比较认可学校，说江湾是凉城地区最好的初中了！



 

7.     复兴实验中学

招收班级：3

 

对口小学是虹口区外国语第一小学（注意不是上外附小哈），建校时间不算长，前身是水电路中学和广灵中学，合并了，加入了复兴集团，学校硬件还可以的，毕竟新建嘛。总体的应试水平基本在区内属于中后水平的样子，虽然学校号称区内联考一直公办第一！我也不知道这个数据怎么来的！说一下家长和实地探访的反馈：

1.     学校在小区里，乐爸去的时候因为还没放学，先咨询了附近的居民，结果附近的居民对这个学校恨得咬牙切齿，原因是学校太吵（额，带点主观了），理由是学校有个鼓号队，有时候会去参加比赛，结果天天一大早就嘀嘀嘀的吹打起来了！居民说吵的完全没法睡觉，都去投诉过好多次了，还有就是学校的喇叭，真的响彻天际，天天被眼保健操的声音吵，额，这个也其实没办法，劝大家以后别买小区内有学校的房子，只能这样了！

2.     学生反馈：学校伙食太难吃，有时候还会有钢丝球的铁丝（额，这个事情好像挺难避免的，乐爸以前学校也吃到过），然后学校老师其实还可以的，但学风比较差，这个学生的反馈说学校有个好班。

3.     学生反馈：学校的体育课真的折磨人，他们老师主要就是练跑步，耐力跑（这个乐爸认为没有问题，学生带了点主观想法）

4.     学校老师的教学方法过于老派，不过好处也有的，学校之前有每周一节课的3L学习，还有一节阅读拓展课，聊胜于无了！

 

 

 

8.     钟山初级中学

招收班级：5

 

对口小学凉城三小，凉城四小，学校是虹口最大的学校了，硬件上可以的，总体数据在虹口排名中游，没江湾好。直接说一下实地采访的情况吧！

1.     学校有好班2个，ab班，a班的市区重点高中率在90%以上，b班未知，这两个班是学校的拳头，师资什么的都是骨干教室，现在不知道还有没有！

2.     家长反馈说小门很重要，如果班里的随迁子女多，那么就有点亏了，比如附近小卖部的老板，理发店的娃也在里面读书的！

3.     学校总体情况还可以的，但如果你的目标很高，那么就有点尴尬，因为学校的特点是缺少尖子生，理念的区重比例还算可以的，但到了市重点这个级别基本到北郊，到华一封顶了，总体缺少竞争的气氛。

4.     学校活动不少，特别是低年级，艺术，体育类很多。

5.     以前虹口这块有个什么政策（叫跨联合招生地块回户籍所在地的政策），现在不知道有没有，招生办的人说江湾和凉城属一个地块，所以以前有不对口的，学校承诺ab班可能会进去，但目前反馈要求高的家长比较后悔。

6.     我是以学生家长询问的，有家长和我说能去摇号就去摇，反正大概率回得来，如果是上海户籍的话！


 

9.     丰镇中学

招收班级：6

 

对口小学是丰镇一小和红旗小学，心疼一下红旗小学（嘤嘤嘤），总体属于区内比较靠后的中学了，17年的时候和附近的新市学校合并了，两个学校半斤八两，但从老数据看，至少新市还有预录取，丰镇预录取好像几乎为0，合并后两个学校的学生还对立过一段时间，额。

说一下家长的同学的反馈吧：

1.     总体教学质量一般，学校并不想外界说的那么不堪，部分老师挺好的，采访同学说特们的英语和语文老师很nice，但语文老师呵呵哒，只会罚抄，作业批改也很慢。

2.     学校操场挺大的，疫情前还对外开放（晚上放学后），学校有管乐团，活动还不少。

3.     放学时间4点半的样子，初三会加一节课，放学的情况乐爸观察下来，学生层次不齐，有打闹的，有约着去玩的，有放学爸妈来接着去学习的。

4.     初三的时候中考人数不算多，好多都因为没有积分，回老家了，红旗小学的生源其实还可以的，毕竟红旗也是老牌虹口优质小学，特别是几个红旗小学毕业的爸妈的理念还是可以的，但从交流的过程中可以看到他们的无奈。

总结一下，学校目前还是比较靠后的，以后待观察！


 

10.   培华学校

招收班级：1

 

就一个班级！额，九年一贯制，菜小+菜中，体育学校，各种体育运动特色！排球、乒乓球、羽毛球、击剑、田径、体操、射击、游泳、柔道、武术、举重、自行车、棋类。。。

作业很少，很多走体院路线的，学生素质嘛，呵呵哒，反正乐爸门口看到不少早恋的，粗口的，额！老师管的很松！
