1. 计算特训：

2. 题型模板化：


· 应用题建立"问题-已知量-公式"对照表（如路程问题固定公式：速度=路程÷时间）

小学数学应用题需要建立"问题-已知量-公式"对照表（如路程问题固定公式：速度=路程÷时间），可直接套用解决80%的典型应用题：

（1）行程问题


问题类型已知量核心公式关键点
相遇问题两地距离、速度和、相遇时间总路程 = 速度和 × 时间注意单位统一（如米/秒 vs 千米/小时）
追击问题两地距离、速度差、追击时间总路程 = 速度差 × 时间被追者先出发的距离需计入初始差距
环形跑道问题跑道周长、两人速度、相遇次数相对速度 = 速度和/速度差第一次相遇时间 = 周长 ÷ 相对速度


（2）工程问题


问题类型已知量核心公式关键点
单独完成工作工作总量、工作效率、工作时间工作量 = 效率 × 时间默认工作总量设为1或最小公倍数
合作完成工作两人效率、合作时间1/总工作量 = 1/效率₁ + 1/效率₂常见陷阱：求合作效率时勿相加错误


（3）利润问题


问题类型已知量核心公式关键点
利润计算成本价、售价、利润率利润 = 售价 - 成本价注意区分“利润率”（利润/成本）和“销售利润率”（利润/售价）
折扣问题原价、折扣率、现价现价 = 原价 × 折扣率“打八折”=80%，而非降价80%


（4）分数应用题


问题类型已知量核心公式关键点
分数增减问题原量、增减百分比、增减量新量 = 原量 × (1 ± 百分比)注意单位“1”的对应（如增加20%是原量的1.2倍）
混合比例问题部分数值、总份数部分数 = 总量 × (部分份数/总份数)常见题型：浓度混合、药水配比


（5）几何图形


问题类型已知量核心公式关键点
长方形面积长、宽面积 = 长 × 宽周长公式：(长+宽)×2
圆的周长/面积半径或直径周长 = 2πr；面积 = πr²注意单位换算（如直径=2r）
三角形面积底、高面积 = ½底×高海伦公式：已知三边时可用 S=√[s(s-a)(s-b)(s-c)]


（6）统计与概率


问题类型已知量核心公式关键点
平均数问题总和、数据个数平均数 = 总和 ÷ 数据个数注意加权平均数的计算
可能性大小可能事件数量、总事件数量可能性 = 可能事件数 / 总事件数概率问题常考“公平游戏设计”


使用方法

  • 快速定位：看到题目先判断属于哪一类问题（如“甲乙两人从两地相向而行”→行程问题）；

  • 提取关键量：用红笔圈出题目中的数字和单位（如“距离200米”“速度5m/s”“相遇时间10秒”）；

  • 套用公式：将已知量代入对应公式，检查单位是否统一；

  • 验证合理性：结果是否符合生活常识（如“速度超过光速”→明显错误）。


示例实战

题目：小明以每分钟50米的速度步行上学，15分钟后发现忘带书包，爸爸骑自行车以每分钟150米的速度追赶，问爸爸需要几分钟追上小明？


（1） 定位类型：追击问题 → 对应第二模块；


（2）提取已知量：


· 小明速度 = 50米/分，提前出发时间 = 15分钟；


· 爸爸速度 = 150米/分；


套用公式：

总路程差 = 小明速度 × 提前时间 = 50×15 = 750米；

相对速度 = 150 - 50 = 100米/分；

追击时间 = 总路程差 ÷ 相对速度 = 750 ÷ 100 = 7.5分钟；


验证：

爸爸用7.5分钟骑行距离 = 150×7.5 = 1125米；

小明在15+7.5=22.5分钟内行走距离 = 50×22.5 = 1125米 → 结果合理。


注意：遇到复杂题目可组合使用多个公式（如工程问题+比例），并养成画图辅助理解的习惯！



· 常见图形题背记"特征值"（如等腰三角形底角=（180°-顶角）÷2）

以下是小学数学中常见图形题的 "公式" 快速记忆清单，掌握这些固定数值和比例关系，解题效率能提升50%以上！


1、平面图形


（1）三角形

图形类型特征值典型应用场景
等边三角形内角均为60°，边长比=1:1:1计算角度、周长、面积
等腰直角三角形角度为45°-45°-90°，边长比=1:1:√2直角边求斜边、面积计算
30-60-90三角形角度为30°-60°-90°，边长比=1:√3:2特殊直角三角形比例关系


（2）四边形

图形类型特征值典型应用场景
正方形四边相等，对角线=边长×√2求对角线长度、面积
长方形对边相等，对角线相等周长计算、勾股定理验证
平行四边形对边平行且相等，对角线互相平分判断图形性质、面积计算


（3）圆

特征值公式/关系典型应用场景
周长与直径比周长=πd 或 2πr已知直径求周长
面积与半径平方面积=πr²半径变化时面积比例计算
同心圆半径差大圆半径-小圆半径=R-r环形面积计算（S=π(R²-r²)）


2、立体图形


（1） 柱体

图形类型特征值典型应用场景
圆柱（直立）侧面积=2πrh，体积=πr²h已知周长求半径、表面积计算
圆锥体积=1/3πr²h与圆柱体积对比（1:3关系）

（2）组合图形

组合类型特征值典型应用场景
圆内接正方形正方形对角线=圆的直径（d=√2a）已知直径求正方形边长
正方体内切圆圆直径=正方形边长（d=a）已知边长求圆的半径

3、图形变换

变换类型特征值典型应用场景
图形旋转旋转角度为90°时，对应点到中心距离不变验证旋转对称性
图形对称轴数量正n边形有n条对称轴正五边形有5条对称轴

4、高频易错点


（1） 等腰三角形陷阱


· 错误：认为底角一定是锐角（正确：顶角可以是钝角，此时底角为锐角）


· 例：顶角100°的等腰三角形，底角=(180-100)/2=40°


（2）圆周率取值


· 计算器考试中π取3.14，手算题可能要求精确值（如πr²保留π）


（3）单位换算


· 平方米↔平方分米：1m²=100dm²


· 立方米↔升：1m³=1000L


5、记忆技巧


（1）口诀法


· "等边三角三等分，角度六十记心门"


· "圆锥圆柱体积比，三倍关系要记准"


（2）图像联想


· 等腰直角三角形想象成直角梯形的一半


· 圆内接正方形画对角线即得直径


（3）对比表格

图形周长公式面积公式
长方形(a+b)×2a×b
正方形4aa²

实战示例：

题目：一个等腰梯形的周长是20厘米，上底和下底之和是12厘米，求腰长。

解法：


· 等腰梯形周长 = 上底 + 下底 + 2×腰长


· 代入已知：20 = 12 + 2×腰长 → 腰长 = (20-12)/2 = 4厘米


掌握这些特征值后，遇到图形题可以先扫一遍题目，快速定位已知条件和特征值，节省50%的思考时间！
