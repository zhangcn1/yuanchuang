（强贴必转，一叶原创）在新中考形势下，魔都家长如何为小升初进行择区和择校

原创漫漫鸡娃路一叶目开2021-02-23 08:22:08

  前言：今天文章有点长，但把所有小升初需要考虑的精华点和不为人知的秘密都说到了。只要耐心细品，一定会受益匪浅。

    最近文章发布以后，尤其是关于前天那篇小升初重要性文章，引起群内家长热议。所以，今天一叶先生略作出补充，让大家更加清晰地明确，在新中考制度下应该如何为孩子在小升初环节根据自身情况选择学校。

 

   分析之前，不得不先详细说明一下2022年新中考的背景：

    大家都应该知道，今年2021年的新中考总分由650分变成了750分，但是招生录取的渠道还是老政策，多出部分是历史和政治的开卷考，外加理化实验考和跨学科综合。一叶先生略作统计后，发现文科和理科的分值被拉成均等，还增加体育平时分数。但是从2022年开始，750分之余，综合评定正式走上台面。在录取的渠道上，发生了天翻地覆的变化。大家可以看一下如下表格：

 

    从以上表格可看出，老政策中四校的录取比例是自招占60%（自荐是全市竞争，推优是区内竞争）零志愿，也就是裸考，占40%。而八大市重点。自招占40%，其中还是本区学校推优为主（这就是爹的力量）。零志愿占15%，名额分配站15%（名额分配其实一直都有，只是数量少，很多人不关心）。裸考平行志愿占30%。

 

    但是新政策出现巨大的变化。四大的录取比例将取消推优（初中内部操作猫腻被遏制），全部采取统一自主招生形式，也就是全市竞争中，高中自主选择，比例占35%（目前暂定，只少不多）。另外65%，一部分是名额分配到区大概占65%*80%，另一部分是名额分配到校，大概占65%*20%。这个是一个明显变化。名额分配到区，实质等同于现在的零志愿和平行志愿，但考核内容不再单单是中考成绩，而是中考的750分加50分综合评定（先根据中考成绩实行1：2投档，然后加上综评分一起排序，择优录取。）

 

    而本来有少量照顾当年招生优待所在区的名额分配到校，也是平均分配到各自区，至于给哪个学校，可能就说不清楚了。这个数量有限，实际一个区也就是1-2人。几乎可以忽略不计。变化最大的还是八大市重点。自招的名额减少到了15%，也是全市竞争，同时取消零志愿，取而代之的是名额分配到校（65%*70%=45.5%）和到区（65%*30%=19.5%）。另外裸考只有20%。

 

    最重要的：名额分配到校只有不择生源的初中才可以享受。而选择生源的初中只有拼自招和名额分配到区，以及裸考。所有以考核选拔录取都是选择生源，而摇号，对口，统筹的都是不择生源。（注意：小学如果选拔，初中直升的话，也算选择生源。）

 

   而八大和市重点，和以前相比，更加倾向于本区。45.5%的名额分配到校让本区的学校将有更多的机会入学。但是，从此竞争将从本校开始。以前，大家都掐尖去民办，然后，大家一起出来抢自招，抢裸考。而现在呢，不用抢了，一半名额在校内，以中考分加综合评定为依据，校内排序的拔尖，就可以直接去八大和市重点。

    那对于校内竞争不过的怎么办？这里还有个名额分配到区的概念。但四校名额分配没有本区的概念，所以我们戏称：四校是全市的，八大是各区的。而八大市重点的外区名额占90-95%，本区占5%-10%。所以，名额分配到区录取，大部分都是去外区学校的。如果都轮不上那时候再出来裸考。裸考只看中考，不看综评。但裸考环节，四校名额是零，（现在FF和SZ给本区还有少量名额，以后就没了。） 而八大市重点只有20%，到那个环节的话机就很渺茫了。所以，以后如果校内竞争不过的，绝大多数学生基本都只有去区重点了。

 

     因此，未来名额分配才是八大和四大的最重要的录取渠道。但是一叶先生发现一个问题，那就是四大是全市的四大，而八大是本区的八大。现在的老政策，四大有推优制度，名额倾向于自己的儿子和干儿子，但以后就并非如此。四大会根据考生人数比例，把大部分名额平均的分配到各区，由各区按照中考加综评的成绩来择优录取。也许只有自招才会顾及到自家儿子。

 

     这就是新中考的改变。对于幼升小和小升初的家长，有必要提前知晓。事实上2018和2019小升初的家长，很多并不了解该政策，还一头雾水的冲去各大选择生源的民办。让我们三四年后一起观看那场激烈和悲壮的拼杀吧。

 

      综上所述，我们总结一下新中考政策最大的改变在于：

1.自招名额大幅少了，大概比以前少了一半多。自招名额换成了名额分配。很多现在有大量自招名额的初中学校，以后不见得有那么好的预录取成绩。由于很多学生在九年级十月前后已经被签约，那以后初三一模考可能就真的只是模拟考功能了。

2. 市重点裸考名额少了，四校裸考没有名额，八大只有20%，裸考的机会比现在少了一半。

3. 科目多了，偏科生的不归路。鸡血家长又要焦虑了，补课的钱不够，时间也不够。

4. 由于综评的定分，是初中和个人录入，再由你填报的高中学校打分。但和自主招生还是有区别的。在操作上，高中实际并没有决定性的权力，因为750分中考是市招办系统自动先筛选。即使很多高中看重的竞赛娃，一旦中考入不了1：2的档位，同样进不了高中综评的范围。（所谓的高中可以安排考试，就是一些不良自媒体在制造焦虑。）

5. 另外，还存在一个隐藏的变化：中考数学和英语的难度有逐年降低的趋势。对从小鸡血的数学和英语，而轻语文的学霸而言，是非常不友好的。试想八万考生，有一万多人数学能考140分以上，那靠什么去拉开差距？一叶先生觉得是语文，历史和道法，甚至跨学科综合。这绝对是理科偏科生和外语原版娃的痛点。未来专攻竞赛和自招的偏科生，真可能自招失利后，连市重点都考不进。

6. 关于综合评定，说实话，这里面的涉及面太广。素质教育以后真不是句空话。但引发的综评纳分项目的筛选，家长要多走心。因为现在机构和自媒体动不动说就和综评挂钩。建议大家看一下组办方，如果没有教委和教育局在其中的，都是骗人的。综合评定在实际操作上，其实只有前25%的学生可以用到。因为只有大概中考排名前25%的学生能入围1：2的圈子，绝大部分参加裸考的学生，综评和你们没啥关系。所以，中考考分和段位依旧是重中之重。

7.以前老政策是全市竞争+同区竞争，而以后新政策变成了50%的同校竞争（名额分配到校）+35%的同区竞争（名额分配到区和裸考）+15%的全市竞争（自招）。竞争渠道变化，让家长对赛道的选择变得格外重要。

 

    只有先了解政策，才能让自己作出最合理的选择。所以，请大家原谅我花费那么大版幅说政策。接下来，让我们言归正传。到底应该如何在小升初环节择校呢？

 

   按照教委的规定，九年义务学校原则上是不能择校的。一般是对口入学的公办学校。但实际情况并非如此。在政策允许合理合法的环节，至少有以下几种：

 

1.民办，超额报名就摇号。

 

2.上外附中，上海实验，浦外，甘泉外国语，上实东滩，上中东。这几家属于特殊性质的公办。前三者实行考核录取。（为什么他们可以考？到底是不是凭成绩录取？大家自己品，至少没看到他们公示录取学生具体信息，所以咱发表不了意见。）后三者，和民办一样，实行超额摇号录取。语言类学校，高考实际没啥优势，这也是事实。

 

3. 买房落户或人户分离。人户分离的跨区和本区都可以办理后就读公办。现在最大问题是，只要办理租房合同就可以办理人户分离。而且没有年限，只要在特定时间前即可（去年是4月13日）。请珍惜这个机会，那这个几乎可以让每个小升初学生都有一次在幼升小择区失败的前提下，获得重新纠错的机会。只不过可惜的是，有些人觉得无法接受统筹。（这个后续我们会详细说）

 

    以上三种情况的存在，让小升初的操作变得丰富多彩。那对于我们即将小升初的家长，应该如何进行选择呢？

 

    目前看，上海高考的985录取率约占学生总数的5%，一本率是30%，高校率是80%左右（含大专高职和民办大学）。而上海中考高中录取的四校率（含分校）大约占中考人数总数（不含没有高中报考资格的学生）的1.5-2%，其他八大市重点率大约为15%，而高中录取率为80%左右。

 

    所以，一叶先生经常说，没有比较就没有伤害。你们需要先给孩子定一个目标：高中或大学的目标。如果在没有任何外力情况下，把以上您的目标高校和高中比率相乘，就是你们实现目标的概率。

 

    对于一心北清交复宏愿的家长，不是一叶重视的服务对象，因为咱们有自知之明，不够资格。我们群内家长的升学目标为：一本和市重点。由于高考的高校录取比例相当高，所以最重要的对决就在中考。一个好高中直接决定了孩子的大学门槛。以一本为例，高中生中的30%，包涵了17%的八大市重点和13%的区重点。那新中考制度下，谁能进入市重点高中或区重点高中实验班，那一本高校基本也就是稳的。

 

    那怎么才能进入市重点高中和区重点高中实验班呢？那就回到了上述说的新中考政策。四个渠道：自招，名额分配到校，名额分配到区和裸考。

 

     一叶先生一直说，自招考的内容和中考无关，事实上和高考关系也不大。尤其是四校的自招数学题，更加接近奥数和高联一试。所有志在四校自招的学霸家长，请注意一个现实：自招前期准备，没有一年以上时间准备是别想了。如果自招花费了一年，还没有被录取，不仅是心态崩了，还有时间和精力去拼两个月后的中考吗？因此，一叶先生认为，以名额分配到校、到区和裸考的渠道去获取四校八大市重点的机会会更合理。至少，准备内容三者是一致的，时间也更充裕。

     那如何在小升初环节未雨绸缪进入到对应学校，如何进入到对应的区去减少竞争压力，获得更加多的机会呢？一叶先生总结如下：

 

1.如果你以985为目标，那需要进四校和八大市重点的实验班后成功的概率才会大。怎么进四校八大实验班呢？那一定是走自招的，请优先考虑自招高中的儿子学校。比如：目标上中，必须想办法进华育。目标复旦附中，可以报名兰生。每个名校都有几个自己的儿子初中，但儿子也有区别。大妈生的和二妈三妈生的名额数量完全不同。原则上是公办优先于民办，民办优先于旁系。

 

    未来自招名额大幅减少，广撒网模式不复存在，而一定是优先照顾儿子的，毕竟儿子都不够分名额，谁会给外人呢？所以，大家明白前滩房价为什么那么贵了吗？花儿未来也许有的10%四校自招率，这足以让魔都所有家长疯狂。

    进入目标学校高中的实验班，也不代表一定有机会，只不过大概有平均50%以上概率可以进985，如果你失去进入目标学校儿子的机会，基本就告别自招。中考考进学生进不去实验班，而四校平行班大概只有前十名才有机会触摸985的门槛。至于其他市重点，估计平行班也就是最多前三到五名。

 

    所以，以985为目标的同学，必须小升初进四校八大的亲儿子学校，还必须排到校内前5%。要进儿子的学校，有两个办法：如果儿子是公办，那只有学区房入学，在里面靠全家的实力挤掉一些来路不明的人，进入校排前5%。如果儿子是民办，那只有去报名，摇号。然后在里面靠自己的实力，挤掉同校的所有竞争对手，进入校排前5%。（爹都很关心儿子的校内排名）这样才有机会进入爹高中的自招视野。

 

（为什么是5%？老政策是10%，新政策自招减半，只有5%）

 

 

2. 如果你是以211为目标。那高中起码必须是四校八大平行班和普通市重点的实验班，然后在里面拼到不是后50%，那基本211就稳了。也就是说，你至少得考进四校八大市重点。按照上文的说法，自招，名额分配到校，到区和平行志愿裸考，皆是可行。

 

那延伸到小升初阶段，如何选择呢？

 

（1）自招上文已说，这里不再重复。（普通市重点也有自招，只不过有人看不上，他们不参加竞赛，基本是看一模和儿子学校的校排）也优先考虑儿子初中。

（2）名额分配到校，一般来说，这个是新中考中学问最大的。根据中考成绩和综合评定同校竞争。所以，人数多，对手弱的初中机会会更大。

（3）名额分配到区，这个是全区自招生录取后的全区比拼。（等同于零志愿，根据目前情况看，区头牌学校的机会会大）

（4）裸考，全区打乱的来的。一些没爹的二三线初中，没有自招机会，名额分配到区后的二流学生，都会考虑这通道。所以综合考虑学校教学、试卷和作业情况，失去自招和名额分配的头牌民办中段生源必定会死拼。另外，裸考还有借读生和跨区考生的存在，对于二三线学校中后段生源，几乎没有机会以此进市重点。

 

     所以，以211为目标的家长，必须让孩子进入区前三的头牌初中（无论民办还是公办）。这个排名不是看家长口碑，不是看自招预录取率，而是看中考实际排名。（可惜官方不公布，民间的真伪自酌），争取校排前20%的成绩。

 

    又或进入二线公办或二三线民办，争取前10%的机会。又或进入三线公办（统筹学校），争取5%的机会。以此进入市重点高中。

 

3. 如果你是以一本为目标。那高中起只要是任何市重点均可。然后在里面争取前50%，那一本就没问题了。第二条中的四个途径一样适用。另外，如果是通过裸考进区重点和普高，只要达到前30%，一样稳稳的一本。

 

    所以，以一本为目标的家长，只需让孩子进入区前三的头牌初中，争取前40%的成绩.或进入二线公办或二三线民办，争取前30%的机会。又或进入三线公办（统筹学校），争取20%的机会。

 

 

4.  至于一本以下为目标就无所谓哪个初中。只要能考上普高，只要高中不垫底的20%，都有大学读。

 

    但是，以上四点只是纸面上的看法。有以下几个因素会影响到孩子最终的实际升学情况：

 

1. 传说的一线学校，大家都喜欢，条子多，关系户多，教师子弟多。即使中考是公平的，那自招途径是否公平？综合评定又是否公平？

2.  可能存在大量跨区掐尖和人户分离的生源，一旦多起来，会影响整个区的竞争格局。（比如今年传闻中的J区）

3.  有的学校教学宽松，有的学校教学鸡血，四年中会产生巨大的差异。

4. 有的老师认真负责，有的老师天天混日子，四年后也会产生巨大的差异。

……

    事实上还有很多影响因素。所以，一叶先生给予大家一些建议：

 

1. 目前看，一线民办环境略优于一线公办，即使如著名公办理科班s为例，靓丽的四校自招率固然存在，但你们何尝知道班级里余下的那些学霸们四年后的失落。对于那些外区掐尖办人户分离的学生而言，也不一定都是好事。一叶先生看好几届中考，条杀牛，在这种学校将比比皆是。别忘记，你们是被请来拉中考分的，自招是轮不上的。你们自招后，中考只要530、还拉什么分？真正能自招的，只不过是披着学霸外衣的虎娃罢了。

 

     所以，对于没有背景的学生而言，你们要的只是公平。条子多也许是一线民办和一线公办最大的差异。而一线民办能给你们大概率公平。虽然借读生不少。但你不让位，他们也转不了正，四年后只能被逼参加裸考。

 

2. 即使统筹进三线公办并不属于天塌下来的那种。学渣多，学霸少的同时，带来的是宽松升学环境。要知道条子是不屑去这样的学校。那你们的机会就来了。同样的分数和实力，在三线公办获得名额分配到校的机会远大于一线公办。在这里，你们只要学会管好自己，不学坏，努力学习，阳光大道一样存在。

 

3.  二线民办一样略优于二线公办。随着摇号的推进，二线民办的生源一定不如二线公办，但二线民办老师的收入牵扯到分数，他们会更有鸡娃动力。

 

4. 至于三线民办和三线公办的比较，一叶先生更喜欢三线公办。因为三线公办的人数大概率大于三线公办。人数多，拿到的名额就多。

 

（有人问，民办摇号了，投简历，小五班，还有用吗？都没有用了。民办摇电，黄牛和机构推优，pet等证书，都没用了。摇号没有猫腻，只看人品。如果摇号有猫腻，就不会有那么多借读了，何必多此一举？）

    所以，一叶先生给大家的2021小升初择校策略略作调整，具体是：

    至于择区建议：学霸去黄浦，一般去虹口和杨浦，学渣去浦东和闵行

 

    黄浦22%的市重点率和不算太强的同区对手，给学霸带来低分去四校和轻松走大同和格致自招的机会。虹口20%的市重点率，不算多的考生，外加数量不少的市重点，是中段生源的福音。而杨浦属于，四校录取高段区，学霸都走了，中段是否就幸福了？浦东和闵行，属于人口大区，学霸多，学渣更多，市重点也非常多，所以，在浦东和闵行，市重点裸考的分数极低，这就是学渣的幸福区。

 

   总之，远离跨区掐尖多的区，远离条子多的区，远离团课多的区。选择了那些黑色地带，你们可能真的连市重点都考不上了。谨慎，再谨慎！

 

（2019年上海高中高考情况参考表）
