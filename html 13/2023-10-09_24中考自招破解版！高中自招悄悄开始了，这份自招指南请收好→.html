# PART 1 #

中考自招详情


01

录取批次


普高和区重主要看的是裸考成绩，即中考成绩，不属于自招范畴。


市重点，四校八校，和部分市特色高中，主要的录取批次有4个，按先后次序是：自招，名额分配（先是到区名额，再是到校名额），平行志愿。


一般来说，市重的生源主要由3部分构成，其中自招占约10%左右的名额。


0

2

自主招生录取类别及顺序



主要有四类，按先后顺序依次录取：
1. 市实验性示范性高中和市特色普通高中自主招生录取；
2. 优秀体育学生和艺术骨干学生自主招生录取；
3. 国际课程班和中外合作办学高中自主招生录取；

4. 中职校自主招生录取。


学生填报志愿可兼报上述四个类别，前三个类别采用学校和学生双向自主选择1∶1签约预录取方式，学生只可选择三个类别之中的一所学校签约预录取。


学业考试总成绩须达到相应最低投档控制分数线方可正式录取；第四个类别采用平行志愿方式，按学生学业考试总成绩和志愿，从高分到低分依次录取。


0

3

自招流程



中考结束后，各高中学校会发布校园开放日的通知。

在高中学校官网报名，需要到高中参加开放日活动；根据校园开放日活动情况，然后在考试院官网填提前批的志愿，然后参加去高中测试，通过后预录取签约。


# PART 2 #

高中自招详情


01

自招时间安排


 1.中考前自招活动 


初二下期中和期末时，分别有些高中会去好的初中，组织摸底，俗称神秘考，或飞行考。最早初二上可以开始投简历了。


初三上的期中考和一模，也有神秘考或飞行考。好学校在抢好生源，好生源同时也在选好学校，双向选择。初三第一学期寒假，有的学校会组织冬令营活动，其性质还是自招。初三二模考后，会有一些自招的动作。


所以中考前的自招，其实没有次数限制的。学校在选学生，学生也在选学校，成绩好的同学，会有很多次机会。这些考试基本都是不公开的，由高中自行组织考试。


 2.开放日活动 


中考结束后中考成绩出来前，是大规模的自招，也叫开放日活动。面向全上海中考生，可以填报上海所有市重点学校。


需要学生自己在高中的官网进行报名，然后参加开放日活动。根据校园开放日活动情况然后在考试院官网填提前批的志愿，然后参加去高中测试，通过后预录取签约。


0

2

自招考试内容



按科目来说，主要考语数英理化5大科，越好的学校，考的科目越多，四校八校和好一点的市重点高中，5科都会考。也有的只考三门，具体看学校。考试内容也会涉及高中和竞赛的知识。


自招考试的，命题，组织，阅卷，完全是市重点高中自己负责，所以每个学校题型和内容都会有差异，难度不一，基本都是跟着四校八校走的。


# PART 3 #

高中自招规划


简历首先是不能少的，简历在初二下学期就可以准备起来了，并在初二升初三暑假前后就投递。校排在每次校内大考结束之后及时更新，并再次投递。目标是四校分校和八大的家长可以在初三寒假投递。


不同初中飞行考形式有差异。最主要的就是一模考成绩和区排名，需保证自己的年级排名至少稳定在前30，时间越靠后的大考成绩越有参考价值。


如果不确定要不呀自招，有机会了就去试试，心态放好，反正没什么坏处。如果基础可以，可以按照自招的难度和要求去学一两科，即使自招不成功，在掉头考裸考中考的东西，自然会觉得很简单。


# PART 4 #

说在最后


自招最开始的一批其实从初二下已经开始了，目标自招的学生即使没机会，简历也要提前准备起来。初三的飞行考和每次大考都是重要的时间点，要重视。


自招属于是顶级牛娃的战场。家长和考生要正确认识到自招不再是必须，名额分配、平行志愿足以上个好学校了。
