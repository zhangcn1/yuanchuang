关于湾区华裔牛娃美本申请惨败后的思考

美本申请失败案例

上周，在美国湾区家长中，最火的新闻应该是一位叫Stanley Zhong的男孩。这位亚裔男孩就读于湾区Gunn High学校（硅谷顶级公立高中之一），履历十分优秀：

SAT 1590，GPA (UW/W): 3.97/4.42；

进入谷歌编程比赛半决赛；

麻省理工 Battlecode比赛全球第二，美国第一

创建了一个做e-signing的startup（RabbitSign.com）；

建立一个非盈利组织教低收入孩子编程；

学校竞赛编程俱乐部的主席；

美国National Merit Scholarship 入围（前1%）。

这么牛，应该会被名校抢着要！

学术方面，SAT接近满分，GPA 3.97/4.42也很高；非学术方面，有领导力，有社会服务，有成功的创业，而且他的课外活动很多都紧紧围绕着他所申请的CS（计算机）专业，可以形成一个完整自洽、令人信服的故事。我相信孩子的Essay，乃至整套Profile绝对是非常有竞争力的。

遗憾的是，名校不仅没抢，还纷纷拒绝了他。

上半年申请大学时，Stanley被16所学校拒绝，不管是大藤，还是便宜大碗的加州公立，都以悲剧收尾，包括MIT、CMU、Stanford，UC系学校。只有UT Austin和U of Maryland两所学校给offer。

后面故事还有反转。前不久，这位男孩被Google L4破格录取，去当高薪（27w+美金）码农了。谷歌的L4级别工程师并不是入门级，而是需要数年专业经验的；本科和研究生入职谷歌往往只是低一级的L3。

背后原因分析

引起大家热议的原因主要有两点：1、这孩子高中毕业就被谷歌录取，是真牛；2、但是这孩子这么优秀，为什么会被16所名校拒绝。

对于让娃走体制外的父母来说，第二点尤其让人费解。

我一位在美国湾区的同学给我做了一些分析，挺有启发，分享给大家。

第一，这个孩子在高中期间不仅做出一个成功创业项目, 还运营着一个非盈利组织，这两项就可以直接工作，大学招生官也会评估，都这么牛了，学校还能给他提供什么；UCSD、UCD、UCSB类的UC系统学校估计觉得他over-qualified，怕给他发了offer，他不来。

第二，他的GPA没有显示学校排名，尤其weighted真的不高，UC 画一条线，可能也进不了；美本比较注重GPA、校排，这也是为什么在上海经常会看到有学生吐槽上中国际、星河湾等卡GPA，在高中时转学其他学校去降维打击。

第三，CS专业人才济济，千军万马过桥，想要被录取的确很难；

第四，牛娃的爸爸是谷歌员工，被破格录取可能是近水楼台，并不适用其他孩子。

当然，亚裔身份在美国大学申请上的确太弱势了。近些年，加州大学取消SAT，亚裔录取比例越来越低。有一个现象叫“亚裔惩罚”：因为申请人是亚裔，大学招生官会把SAT分数降低几十、甚至上百分，再去评估是否录取，而有些族裔，则要加上几十甚至上百分。

普林斯顿曾有研究发现，亚裔相对于非裔，要接受280分的“SAT惩罚分”：其它条件都相近的情况下，考1550的亚裔和1270的非裔申请人具有同等的录取机会。还有的研究指出，这种惩罚高达450分。

申请经验学习

美国的大学入学主要是由AO(admission officer)决定，而非国内的高考制度，比起讨论这位华裔高中生为什么不入各位AO法眼，我觉得更应该关注为什么这位高中生能够走捷径直接入职硅谷大厂，尤其是在今年，很多名校的CS毕业生都找不到工作的情况下。

如果仔细看一下这位高中生的简历，就不难发现，他有一份非常重要的经历：参加过Google Code Jam，一个由Google发起和组织的编程竞赛。其实硅谷和华尔街的公司历来有赞助青少年竞赛的传统，如果大家去看一下IMO, IOI, ACM-ICPC的赞助商，不难发现到处都有这些公司的身影，赞助这些活动对公司而言也是最好的招聘渠道，在这位华裔高中生之前，Google和其它一些硅谷大厂一直就有招收高中生入职的传统。

（Google Code Jam历年获奖者名单）

Google Code Jam是2003开始由Google举办的给全世界编程爱好者的比赛，至今已经举办20年，疫情之前，每年的前25名选手会被邀请到Google总部进行决赛。进入到半决赛甚至淘汰赛的选手，都有机会拿到Google的面试机会，这个华裔高中生就是进入到了半决赛从而拿到面试入场券的。

由于这个比赛和信息学奥赛比较类似，在众多的得奖者中，我们可以看见很多中国大神的名字，比如著名的第一届姚班的楼教主(Tiancheng Lou)，曾获得2008, 2009年两届冠军，同样来自中国的Lingyu Jiang获得了2022年第二名的好成绩。可惜的是，Google Code Jam因为各种各样原因，2023年是它的最后一届，明年将不再举办了。虽说Google的不举办了，但是还有其他的编程比赛，比如Meta(Facebook) Hacker Cup，这些都是很好的大厂工作敲门砖。

虽说高中生要不要跳过大学直接上班是个见仁见智的问题，但是至少在美国，比起卷藤校，这也是一条收益不错的路。我一直认为，比起万人卷奥数，信息学奥赛的ROI被严重的低估了，随着人工智能的崛起，信息学奥赛即使搞不到国集水平，将来也能有一份不错的收入，性价比杠杠的。

教育反思

当然，在鸡娃的战略战术上，不管国内还是美国，大家都技艺愈发精湛了，如果要比技艺，是比不完的。不谈这位亚裔高中生在申请过程中，孩子爸爸利用自身优势对其过度包装，过度托举，但当孩子面对自己的大学申请、人生困境和采访机会时，家长完全可以在现场台下支持，把舞台留给孩子自己，让孩子独立发声。

但我看到，整个admission申诉过程，都是孩子爸爸在发声。娃是个coding高手，难道社交媒体不会用？作为start up创始人，难道自己的事情说不明白？已经18岁的成人了，应该是有能力表达自己的观点，甚至可以自己向教育部申诉了，为什么还是他爸在发声？在被电视台采访的时候，为什么孩子爸爸陪在一旁？亚裔平权道长且阻，娃不是我们的second life，共勉。
