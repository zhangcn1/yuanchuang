校内成绩：网课期间无排名，估计中等偏上。
下学期目标：班级排名前茅。
小升初目标：冲刺一梯队，希望保底二梯队。
目前学科课外配置：
语文：线下阅读精读课与小笼妈语文小组，总计1周2次约4小时；

2. 双职工家庭，有时加班。一般情况下妈妈19:30到家，爸爸20点后到家。（年后职场搬迁，妈妈可能需要20点后到家，焦虑中）
目前妈妈下班后负责与孩子沟通时间计划，布置、批改作业。二上网课期间由于无人看管，上课质量大打折扣。


01
—

寒假计划



寒假作息与网课期间作息相同，每天6:45 – 7:30晨读，网课期间已完成部分作业，寒假每天具体课业时间安排与孩子在前一日总结当日学习情况后进行制定。
目前寒假计划的学习内容除了预习和复习，与居家网课时差异不大，也在考虑是否需要进一步调整。（在此也恳请大家给予建议）
（以下计划排列顺序为优先级顺序，各科均有调整的空间）

一、语文：每天约3小时
1. 二上复习 / 计划20分钟
每日自行复习一课，晚上使用飞翔字词作为默写进行复习。
2. 二下字词预习/ 计划30分钟
两天预习一课，使用绘本课堂进行字词预习，并且对课文进行预习精读。该方案预估可以预习二下大部分课程，可能根据实际情况再调整，比如把字词预习涵盖二下所有课程等。
3. 基础知识/ 计划10分钟
复习总结已完成的二年级基础知识的错题，加深巩固。
4. 泛读 / 不得少于60分钟
每日小说阅读量与孩子约定不得少于60分钟。多多益善。
5. 阅读分析 / 计划平均45分钟
结合寒假班精读课程与小笼妈课程，再使用《学会阅读》每日一篇进行巩固。
6. 古诗词 / 计划15分钟
晨读时巩固复习小学生必背古诗词，完成一篇《小学生古诗词阶梯阅读》。
7. 作文 / 计划平均15分钟
每周2-3篇日记或者看图写话，或者喜欢的故事续写，看孩子自己的意愿。

寒假期间语文的重点主要是二上复习+二下预习，其次是养成泛读习惯。实操后如果效率不佳，可能会舍弃作文练习，并后续不再要求补做。
二、数学：每天约2小时
1. 小奥课程 / 计划平均 90分钟
1）数学进阶小组：每周参与数学小组的活动。
2）复习天天练之前学过课程，做对应的奥精、奥数教程等题目。
3）天天练：根据孩子实际情况自行安排感兴趣的章节，每天1课。
2. 计算题专项 / 计划 30分钟
1）学而思计算秘籍，四年级/五年级。
2）三招过关，四年级。
3. 预习 / 弹性计划
目前该部分计划不清晰，之前暑假预习到四上（使用课本 + 一课一练），考虑到寒假时间允许的话，结合天天练或者其他材料继续预习。由于尚未实施过自行预习数学的经历，故此部分暂定为弹性计划，可能只是春节或者周末的时候实行。
寒假期间数学的重点主要是跟上数学进阶小组 + 奥精等复习，其次是计算题的练习。天天练与四下预习，根据实际情况会维持或者调整减少学习量，根据实际吸收情况制定合理的学习计划。

三、英语：每天约1.5小时
1. 单词默写，每天复习+新增30个单词复习。
之前有用常青藤爸爸KET系列丛书打卡，但是随着难度的上升，明显有些吃力。目前还是从小学生单词开始稳扎稳打。
推荐墨墨记单词，无痛记单词首选，单词库也较全。
  
2. 英语课程 / 计划60分钟
目前是SBS与新概念同步上，一个偏重语法，一个偏重单词。上课以及加上每次课后的作业，折算每天大概需要1个小时。
3. 英文书阅读 / 弹性计划
主要以兴趣为主，翻阅感兴趣的绘本，不做硬性要求。
4. 辅导书 / 计划30分钟
全新阅读 + 听力，每日一课。


寒假期间英语的重点主要是培养英文绘本的自我阅读，其次是完成对应SBS与新概念课程的作业。如果阅读习惯养成的话，可能减少教辅的作业量。

四、其他：每天约3小时
1. 素质课程：以围棋课为主，一周2-3节，每次3小时。不上课的日子里每天做题+下棋，预估在1小时。
2. 体育锻炼：每天不少于30分钟。
3. 兴趣阅读：由于幼儿期间比较喜欢科普类书籍，故家中该类书目也较多，因此每天安排30分钟自由阅读时间。

02
—

学习情况



目前对孩子的学习上比较担心的是语文字词的积累，与阅读习惯的养成。因此寒假第一优先级是一类词、二类词的复习与预习。其次目前偏爱口水书，寒假养成阅读小说的习惯也是主要目标。

一、语文
1. 基础知识
识字比较早，家里也没有培养，自己潜移默化中就认识了很多字，但正因为家里没有培养，虽然会认字但是对词语理解不足。寒假着重强调复习和预习，暑假的时候默写没有坚持，只是坚持了抄写生字，实际最后发现是不足的。因此这个寒假将强化校内基础知识部分。
2. 阅读
目前采用精读 + 自己做辅导书的方式，考虑后续如果精读效果吸收巩固的话，自己做辅导书的作业量进一步减少。
泛读上，逐步过渡到阅读大部头小说，而非口水书。
3. 古诗词
寒假考虑把小学生必背古诗词复习熟练后，二下开始进入默写阶段。逐步在二升三的暑假加入古文的学习。
4. 作文
作文之前关注度较小，暑假的时候有坚持每周写俩篇日记或者看图写话，但开学后没有坚持。寒假里准备和孩子一起学习，逐步过渡到对词句进行精修，提升作文的质量。

二、数学
大班期间由于搬迁等事宜，导致父母精力无暇顾及孩子，因此课余时间都是随孩子性子看书、下棋。突然有一天发现本来只是会两位数加法的孩子不知道从哪里就学会了竖式、除法，当时甚是惊喜。
直到现在数学也是孩子最喜欢的学科。不过也遇到一些瓶颈，自从看了睿爸推荐后使用天天练至今，原来可以自己学习2-3课/天，基本准确率都是三颗星。但是在三年级数字谜、应用题3等模块，开始逐步缺少耐心，没有妈妈陪的情况下，不太能够自己完成课后练习。因此寒假期间加入了复习模块，希望在同样的课题下巩固后有所提升，也在这个过程中性子更稳重些。

三、英语
从2岁开始启蒙送到双语幼儿园，但是疫情期间停了外教也停了英语课（最错误的决策）。上了小学一下子回退到RAZ B，妈妈的信心也是打击殆尽，迟迟没有恢复。一升二的暑假开始重拾英语，SBS学习到第三课，新概念L35课。老师反馈都说孩子语感较好，学习吸收情况较好，希望可以继续保持，稳扎稳打。寒假期间线下老师也都停课，这段时间考虑用来巩固之前学习内容 + RAZ重拾。并且根据这段时间的表现，来进一步考虑下一阶段的英语学习重点和方法。（欢迎各位家长指导）


03
—

踩过的坑



一、语文
1. 字词：以为认识了就是会了，认识字、会写绝不等于懂。直到孩子升二年级，妈妈才开始幡然醒悟，开始用《语文知识周计划》学习语基。
低年级阶段字词才是基本功，值得费时间费妈。

2. 泛读：幼儿阶段都是随便孩子看什么书，喜欢看什么看什么，前期看了些科普，后期看了一些口水书，没有养成看大部头书的性子，并且也没有从书中获得共情的能力。
随着孩子越来越大，也越发意识到共情通感的重要性，阅读时候如何正确想象，如何置身其中，如何换位思考，除了阅读方法的本身也依托大人日常对孩子思维模式潜移默化的影响，输入和输出，理解和沟通是相辅相成的，而只聚焦某一方面，可能会陷入顾此失彼的尴尬困局。
阅读需要看孩子兴趣，也需要正确的指引。

3. 书写：虽然从大班开始练字，但是没能坚持，导致到二年级还是狗爬字。书写的弊处不仅影响学习效率，而且在校内学习过程中，常常卷面扣分、写字本多次返工。

二、数学
天天练三年级前的数学大部分让孩子自我学习，当时注重答案对错，没有养成合适的学习习惯，导致现在没有养成规范步骤、完卷检查等习惯。试图让孩子加强这俩点，又有排斥心理。
另外可能从小围棋学习的加持，孩子在计算方面起步较早，而数学的启蒙也往往从算术开始，因此算术上的一些细小的成就往往会让家长对孩子在整体数学体系上的能力产生误判，在逻辑、几何等能力的培养上往往是最容易被忽视的，这些能力同样需要更早更准确的一些引导。
好的学习习惯需要从小养成，在看孩子学习成果的时候不仅要看对错也要看过程。

三、英语
如上文赘述，语言学习细水长流，家长不可放松。一时的放松，换来的只能是无休无止的追逐。

四、兴趣爱好
幼儿阶段曾经尝试过足球、羽毛球、网球、游泳、围棋、大提琴等，从一而终坚持下来的只有围棋，孩子现在课余时间也愿意花时间在围棋的对弈上。但一路走来，大提琴、游泳、围棋都有鸡飞狗跳的过往，而之所以坚持下来的是围棋，或者与其说只对围棋产生了兴趣，其实更本质的是孩子还是更容易也更喜欢在一个个输赢胜负下获得的成就感。每次成功升级升段以及在集体中获奖发自内心的喜悦对孩子来说都是最难忘的。
兴趣爱好需要长远规划，广撒网的同时也要聚焦，及时止损，不然沉没成本会越来越高。

五、作业在于精而不是量
一升二的暑假，由于疫情，是孩子第一次没有文化课的暑假。于是妈妈布置了很多作业，在22年8月末第一次接触睿爸小屋的时候，才意识到自己重复性作业太多了。家庭作业还是求精不求量。
一升二暑假期间完成的部分作业

04
—

写在最后


陪读的日子里有放松有紧盯，有欢笑有争吵。作为家长，体会到几个误区：

1. 即使是小学作业，也并不简单
父母虽然都是数学系，但对奥数并没有太多的了解和接触，说实话三年级的奥数还能应付，四年级已经有些慌张。更不要说作为在学生时代主修日语而非英语的妈妈，在英语上的指导已经捉襟见肘了。
责怪孩子蠢笨之前，先想想自己真的会吗？

2. 不要攀比
家长有时候会迷失在各个群里，为什么小A考了满分？为什么小B英语已经看了原版书？为什么小C参加比赛得奖？都是一样大的孩子，为什么自己家的孩子落后？
家长的焦虑最终反映在与孩子沟通的态度、给孩子布置的作业上，不仅让家长越发焦躁易怒，也要孩子无力承受。
横向比较在某些时候是需要的，但更关键的是纵向比较，自我提升更为重要。家长更多的可以从去看其他家长的建议去及时调整自己的路线方案，而不是盯着孩子与孩子之间的差距。
实话说现在看睿爸的每日精选是妈妈的必备功课，从中不仅可以学习其他家长的经验，而且会有适龄的书本、教辅推荐，可以说大大的降低了试错成本。

3. 孩子模仿的是家长
在要求孩子这样那样的时候，有多少家长是在玩游戏，有多少家长是在刷视频。孩子模仿的是家长，如果家长在家里无法作为一个榜样，那么是否反思下对孩子的要求过高？

4. 孩子不是父母期望下的奴隶
曾经也迷茫过，面对付出的时间、精力、金钱，会要求孩子达成某种目标，如果没有达成会失望、生气。但孩子作为独立的人，过度的打压可能会造成他无法塑造独立人格，也可能产生厌学想法。
兼职做父母，全职做自己，给彼此一些空间，这样我们才能共同成长。

感谢在升入二年级的时候遇到睿爸小屋，在迷茫纠结的学习路上给家长启示与方向，希望新的一年，可以与孩子一起共勉。


小屋群管建议：


1. 总体建议：

  • 语数各科反复复习，更多的是不笃定，害怕自己前期基础没打好，后期会塌，要补各种漏洞。建议妈妈放开手，往上面冲，别总担心。

  • 三门学科看来，英语的学习规划问题最大。下面的学科建议请家长自行平衡选择。

2. 语文：

  • 每天复习一课，太慢。预习两天一课，更慢。这样太拖累时间。复习抓重点，预习不是学一遍，抓不熟悉的。建议复习每天1~2课，预习每天1课。

  • 计划中1.2.3都可以用《周计划》一年级和二年级本代替字词预复习，拓展课文外的语基知识。

  • 二下的课文不需要提前精读，课堂跟着老师就行了。

  • 目前小低年级还有时间，可以多些语文素养类的尝试，泛读的书不拘于小说类，可参考学校书单或部编教材推荐必读选读书目，也可看看睿爸小屋公众号群友荐书活动小低年级篇，会拓宽视野。

3. 数学：

  • 不建议用天天练复习，可以用它预习。进阶小组的知识点如果没有学过，可以提前梳理一下。

  • 天天练校内或者思维，1课不需要花很长时间，寒假时间长，可以考虑每天多学习几课。并通过练习去加强对知识点的理解和掌握。

  • 数学之前练了不少，但梯度不够，都是在比较浅的内容上面打转。希望通过进阶小组能摆脱这种情况。

4. 英语：

  • 如果ket词汇都有困难的话，说明英语不管是听力还是阅读都比较弱。以上海英语的考试水平来看，孩子的英语水平远远不能达到小升初目标中的“目标一梯队，保底二梯队”。

  • 建议原版阅读和应试2条线一起抓。孩子目前学英语的方式学科学习比重太多，靠课时堆英语，阅读太少了，raz的B级基础不够开二英。孩子才二年级，现在资源这么多，不建议用低效并且很容易有天花板的方法。

  • 建议多听读，因为英语没有外部环境，所以阅读与听音频对英语学习还是挺重要的。重心从背默词汇转移到阅读上面，先眼熟了再背会轻松很多。英语的听读没有进到高阶桥梁或者初章都不太建议背剑桥单词，全新英语并不太适合孩子目前学习。

  • 建议提高些刷题效率，不用盯着小学英语刷题，小学英语的教辅目前来说对于鸡娃家庭是偏简单的，不用按年级刷听力/阅读/语法等练习册，先积累后再去有针对性的做弱项训练（可直接跳过小学教辅或直接做小升初教辅）

  • 建议英语教材学习+阅读为主，课内词汇与教辅练习先放一边，因为阅读+教材已经能覆盖小学词汇了，再单拎意义不大。
