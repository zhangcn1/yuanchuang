幼儿园女宝的学习习惯养成

原创睿爸小屋睿爸小屋2021-02-28 07:59:00
编者荐语：群友萱萱妈学前鸡娃总结。

大家好，我是幼儿群里的萱萱妈妈。现居吉林4线小城，家里有一个2014年大月龄女孩，正读大班，今年9月上小学，记录一下幼儿园这两年多的鸡娃经历，希望大家多多指教，提供宝贵建议。


01
—

小班


萱萱因为大月龄是近4岁上的小班，片区公立幼儿园，每天就是蹦蹦跳跳，不教任何文化课。孩子上幼儿园之前就是个文盲，语数英都是零启蒙。因为我上班非常忙,姥姥负责看孩子，老人文化程度不高，没能力鸡娃，萱萱每天就是玩和看电视。

上了小班后，正巧遇到调岗，为了能有时间多照顾孩子和启蒙学习，我主动调到了一个工资少但早8晚5还有双休的岗位，就这样开始了鸡娃生涯。我用了一周时间思考加上网找资料，制定了一套我认为萱萱比较能接受的鸡娃计划。

小班计划

语文：天天训练法
1.四五快读（两天一课）
2.一年级字卡（每天认5个字）
3.亲子读绘本（每天读3本）
4.婷婷教诗（每天听两集）

数学：五步法
1. 认识数字1-100，实物，计数器与数字一一对应（1个月）
2. 会写数字1-100，每天写半页（1个月）
3. 反复用计数器和实物练习数的增加和减少（1个月）
4. 加减法口算练习：公文数学每天2页
5. 用50内加减法解决简单的应用题（一本全是应用题的练习册，名字忘了）

英语：动画音乐法
1. 看小猪佩奇，每天一集，第一遍听英语版，第二遍听汉语版。
2. 看小鼠波波，蓝色小考拉。每天各一集，只听英语。
3. 买了个小音箱，每天上下幼儿园路上放英语儿歌。



小结：

刚开始的日子是痛苦的，萱萱太小，之前又没有学习的习惯，一喊学习就满屋乱跑，又哭又叫。好不容易抓进屋里学习，也是一脸懵逼的坐着发呆，几分钟后又跑开了。大约过了一个月吧，情况才有了好转。

萱萱先是喜欢上了亲子阅读，从静静地听故事，到开始有自己的思考，会主动问一些问题，一些简短的故事听多了甚至可以背下来讲给我听。英语动画慢慢也爱看了，但英语儿歌还是不喜欢。数学喜欢应用题，不喜欢算口算，可能应用题更接近生活吧。

虽然有悲有喜，幸好基本按照计划坚持下来了。小班结束，识字大约600，能算50以内加减法（20内算的更准更快一些），英语就没列为重点鸡，只是启蒙，所以也没什么大收获，能听懂一些你好，谢谢之类。我们这3年级起学校才有英语课，所以倒也不急。


02
—

中班


小班时由于太小，怕肌肉力量不足，所以我除了数字1-100，没有安排书写汉字，进入中班后，决定开始练习写汉字。另外准备重点开始鸡拼音，600的识字量加上拼音，争取中班末能够自读有拼音的绘本。

中班计划

语文：
1. 声母和韵母，每天认两个，各写两行（一个月）
2.音节的拼读与声调 ，每天拼读练习一页（一个月）
3.每天写一个笔划5行
4.每天自读带拼音绘本1本，亲子阅读绘本1本
5.日有所诵一年级，小学必背古诗75首每天读10分钟

数学：
1. 学习100以内加减法
2. 每天公文口算两页（100以内）
3. 学习一年级数学课本（上下）中计算以外的其它内容
4. 做百花思维1-4册

英语：
1. 看《巧虎启蒙英语》、《清华幼儿英语语感启蒙》每天一集
2. 海尼曼，RAZ，牛津树，培生等绘本每天随机挑3本，用点读笔跟读
3. 报机构的寒暑假自然拼读课，平时在家每天从绘本中挑出5个单词练习。
4. 每天学习3个单词（BNC常用15000词），反复读，常用的意思讲解一下，试着从看过的绘本中找到这些单词，在绘本句子中加深对单词用法和意思的理解，因为个人英语水平不高，做不到自己造句来给孩子讲解，只能用绘本或字典中的例句。



报班情况：
1.硬笔书法：每周2次
2.画画班：每周2次
3.篮球班：每周1次
4.跆拳道：每周1次

小结：
1.拼音学完，拼读书写还算顺利，大部分无错误。但受地区口音影响，个别音读不准。平翘舌分不清，总是乱读，平舌读成翘舌居多。po，bo，mo读成pe，be，me，yin读成yen。所以家长们如果能说标准普通话，那么自鸡拼音就有优势了。
2. 基本笔画能做到书写工整，就是书写速度慢。
3. 数学100内加减，准确率90%以上，1年级上下册教材内容学完。
4. 必背古诗75基本都会背，没有专门背，读的多的，某一天忽然就自己会背了。
5. 英语能认识26个字母并书写，自然拼读会单个字母的发音和简单字母组合的发音，5个字母以内的单词基本可以靠自拼拼对。GK水平的绘本可以自读。
6. 能看懂简单的英语动画，会唱几首儿歌，常见词能认识600-700左右，能说出一些简单句如“ I like apples.”。


03
—

大班


大班的学习重点放在了英语，语文的阅读精读和写字速度上。数学正常进度往下推。各种补习班照常上，无中断放弃。

大班计划

语文：
1. 每天1页一年级生字（15分钟）
2. 自主阅读每天半小时（图书馆自已选喜欢的书）
3. 每天做一个阅读，错题详细讲解（阶梯阅读一年级）
4. 日有所诵二三年级每天读10分钟
5. 每周一篇看图写画或读书笔记（口述）

数学：
1. 背诵乘法表，用实物演示变化（一个月）
2. 乘除法练习，口算每天一页
3. 上半年学习二上课本，下半年学二下课本
4. 做百花思维5-8册

英语：
1. 读英语绘本每天3本（RAZ E、F，牛津树5、6，海尼曼G1随机选）
2. 英语字帖每天写半页
3. 继续学BNC常用词汇总15000
4. 蛋糕英语课每天1节
5. 复习自然拼读（FREE PHONICS）



小结：
大班过去半年了，按部就班的按计划每天学，语文英语进度略快，数学慢一些，百花思维一些图形找规律的不太行，空间想像力差。专门买了《立体王》，打算下半年好好练一下。


04
—

 谈谈鸡娃经验


我和萱萱生活的小城市，教育不比大城市的“花样百出”，没有幼升小，小升初，不存在单招，摇号，分班考之类，就连私立学校都没有一所，全部按学区上对口公立小学中学，高中呢目前只有一所，将来能考上最好，考不上家长掏个10W+托关系也可以进（之前一直是这样，9年后谁知道呢），不过也就可想而知这所高中的质量了，不难考。

对萱萱而言，中考压力并不算大，目标可以直指高考。我算是小城里比较鸡的家长，周围鸡娃环境不好，佛系家长居多，都觉得时间还早高考太远了不用着急，最多报几个补习班就算鸡了。所以我也只能上网加鸡娃群，寻找来自五湖四海的盟友们，大家互帮互助，抱团努力。

我目前的困惑：我只有这一个女儿，智商一般，学习能力普通。不希望她将来离家太远，所以高考的最高目标就是东三省的4所985：哈工大，吉林大学，东北大学和大连理工，最低就有个本科能读就行。

我个人目前有一个傻傻的想法，就是一直提前学，就算不懂塞也要塞进去，在上高中前学完高考所有内容。然后高中三年就一直复习复习复习，所有内容弄个滚瓜烂熟，能不能靠熟练度考上一所好大学。这个想法肯定是有问题，但我不知道问题在哪里。

都说“取法乎上，仅得乎中”当然要按最高目标努力，那么在可以忽略中考压力的情况下，只考虑高考，不走竞赛和艺术，小初高各个阶段怎样规划各科学习性价比最高（例如，先学哪科后学哪科，哪些可以不学，哪些可以课内深度学，哪些需要玩命拔高学），在此诚恳欢迎睿爸及各位家长朋友们给些建议，先行谢过。

睿爸观点：

1. 鸡娃要定目标，但尽量避免预设结果。

东三省的985录取率约在2%~3.5%之间，一本录取率约在12%~14%之间。
大多数省市从小学到高中会分流约50%的孩子，现在假设小学阶段年级人数4个班共200人。
当目标是东三省的985时，那么孩子在小学阶段要在top 2%以内，也就是差不多年级前4，班级第1。
当目标是东三省的一本时，那么孩子在小学阶段要在top 7%以内，也就是差不多年级前10左右，班级第3左右。
预估这些数据并不是说现在就一定要争取这个名次，而是心中有数，根据实际情况调整。孩子从小学到初中高中的变化会很大，家长和孩子尽力而为，但接受任何结果。

2. 不要给孩子贴标签。

孩子还小，家长不要这么早贴“智商一般，学习能力普通”的标签，更不要让孩子自己有这样的心理暗示。

3. 不要为了提前而提前，避免填鸭式教育。

“就是一直提前学，就算不懂塞也要塞进去，在上高中前学完高考所有内容”
别说高中了，就算初中阶段如果学的糊里糊涂的话，都很难提前一年的进度。学习要看孩子的实际情况调整，没有一定深度（理解）的提前是没有意义的。而且，考好大学也不会仅仅是熟练度的问题。

4.一般来说小学阶段先语英，后数学。请参考小学数学鸡娃路线（6.分年级规划小奥学习）。


关于鸡娃经验，其实我认为学习就是孩子自己的事情，只是年纪小不成熟，父母起一个制定目标，督促学习和答疑的作用就行。娃是学习的主体，家长只需要配合。具体怎么配合，我目前的做法是：

1.  帮孩子认识到学习的重要性。

心灵鸡汤隔三差五来上一碗，平时看到公众号上有这样的文章，就念给孩子听，顺便讲讲妈妈朋友中张三李四不好好学习现在只能顿顿吃白菜之类。至少当天萱萱的学习态度会很积极，偶尔特别感人的文章能起效四五天。

2. 榜样的力量，陪着她共同努力。

我从来不会做自己看电视，却让孩子一直学的事情。孩子写字看书做口算题这些不需要我教她的时间，我会坐在旁边学我自己的专业知识。鸡娃两年多，除了孩子的进步，我自己也收获了很多，考过了会计高级。

3.  尽量不要打骂。

太小的孩子实在听不懂道理时，其实沉默也是一个办法，过一阵气消了，再继续学。另外女孩子总是更脆弱敏感一些，我生怕哪一次的过激行为会给亲子关系埋下隐患。

4.  鸡娃迷茫期可以多看看鸡娃公众号。

尤其睿爸的数学规划系列文，还有一些群友的经验分享，实操性很强，上小学后可以直接COPY了用。这也是我写这篇文的初衷，没谁就有哪位家长和我家的鸡娃背景和目标接近，就可以参考一下嘛。还不行的话就上鸡娃群里把你的问题说出来，总会有热心人解答，碰到真正能解决你问题的答案，记得发个红包，回报一下。

5.  连蒙带骗哄娃。

比如参加打卡活动前会跟孩子说，这是全国培养天才少年的打卡群，里面都是全国最牛的孩子，妈妈好不容易才给你报上名，你不坚持认真打卡，就被踢出去了，你就再也没机会当天才了。

6. 坚持。

学习像烧水，暂时看不到效果放别弃，只要坚持，水总会有烧开的一天。大道至简，千万别决心下的慷慨激昂，结局却是怡笑大方。

2021，砥志前行，不忘初心！
