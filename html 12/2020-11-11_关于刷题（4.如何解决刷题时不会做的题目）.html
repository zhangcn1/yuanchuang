关于刷题（4.如何解决刷题时不会做的题目）

原创睿爸小屋睿爸小屋2020-11-11 07:59:35

刷题时孩子肯定会碰到不会做的题目，本文给一些建议。

对于“什么样的题目需要放弃或者暂不解决”请参考鸡娃系列家长篇(3.家长如何帮助孩子解决不会做的题目）。


本文讨论的科目无特别说明均为数学。

本文讨论的内容为刷题，关于解题和错题另文讨论。


本文适用人群为：

  • K12义务教育阶段的孩子。

  • 希望能辅导孩子学习或者检查孩子作业的家长。


本文适用场景为：

  • 孩子遇到不会做的题目或者做错但想不明白的题目，在已经思考了很久的情况时。



01

—


请教老师


1. 解决问题的效率：很低

有很多家长或者孩子不好意思问老师问题，即使问也不好意思一次问老师太多问题。

大多数孩子问老师的问题一次不会超过3个。


除非是孩子上课时碰到的问题，否则往往不能及时解决。而问题A没有及时解决，往往会影响问题B、问题C的解决。

微信群、私聊老师未必能及时回复。校内的问题，往往要等到工作日才能问老师。培训机构的问题，反馈速度视培训机构整体的服务质量和老师的认真负责程度。

如果孩子的问题不能够当天被解决，很有可能会被搁置。


2. 解决问题的能力：强

老师能解决大多数孩子学习过程中的问题。


3. 解决问题的质量：高

老师在回答问题时能精准、快速的发现问题的核心，能提出更有针对性的解决方案。

老师在回答问题的过程也容易了解孩子现在学习的情况、薄弱点。

如果是共性问题，老师在授课时会做适当调整，有些机构老师会针对具体问题录制视频讲解。


4. 解决问题的耐心程度：很耐心

大多数老师都能耐心的回答孩子提出的问题。


02

—


请教同学


1. 解决问题的效率：低

无论校内校外，孩子相对容易找到可以问问题的同学，且孩子和同学沟通容易，交流压力小。


2. 解决问题的能力：强

随着年级越来越高，同学之间互相解决问题的能力越来越强。


3. 解决问题的质量：不稳定

同学没有老师专业。

同学会做一道题，也许只是因为他以前做过同类型的题目，未必代表他能精准的发现问题真正的关键所在，解决的可能是一道题而不是一类题。

同学对一些细节的理解未必正确，讲解未必规范。

部分同学思维过快，易跳步。

当孩子和同学水平差距较大时，很可能会理解不了同学的“显然有”。

部分同学表达不清会影响孩子对问题的理解。


4. 解决问题的耐心程度：视人而定

回答问题不是同学的义务，同学的耐心未必足够。



03

—


学习群、解题群、鸡娃论坛等


1. 解决问题的效率：很低

孩子先思考问题解决不了，然后家长再思考问题解决不了才会把问题扔到学习群。

问题未必能被有能力解决问题的群友及时看到。

有能力解决问题的群友未必有时间、有兴趣回答这个问题。

家长未必能在群内及时看到群友回复的答案或者解答过程。

群友回复的答案或者解答过程未必正确。

一道并不难的题目经常要折腾好久才有眉目，解决问题的效率并不高。


2. 解决问题的能力：低

学习群内往往不会探讨难题。

大多数情况，学习群内更多的是家长们互相分享心得、交流信息、收集资料，而不是讨论具体题目。即使讨论具体题目往往也是相对简单、基础的题目。

大多数群友毕竟不是专业人士，能解决的问题往往不会很难。相对来说一些专业的解题群群友解决问题的能力更强，但普通鸡娃家长未必能融入这个群体。


3. 解决问题的质量：低

很难得到问题的系统性回答。

对于水平不够的家长来说，把群内得到的解答消化后再给孩子讲解的过程可能存在信息上的丢失。而且仅仅一道题的解答，就学习而言并不系统。


4. 解决问题的耐心程度：视人而定

回答问题不是群友的义务，群友的耐心未必足够。

即使群里有有能力的家长，未必能有那么多空闲时间答题，也未必能有兴趣去解某些题目。

就比如说我看到群里家长提出来的问题，在我心情好有空的情况下：

如果问题比较基础几句话能说清楚的，我会回答。

如果问题比较基础但需要说明的内容太多，我就懒得回答，最多是告诉对方 “请参考天天练X年级XX知识点”。

如果问题较难但我比较感兴趣，预估要花费一段时间，那在我有时间时会做做。但因为这一类问题往往耗时较多，经常会出现我解答出来问题之后，问这个问题的家长已经不知道哪里去了。长此以往，我对这一类问题回答的兴趣也不大了。

如果问题较难，且没有标准答案可以验证，我往往直接略过。比如小奥复杂一点的计数问题，经常算了半天还不知道自己做的对不对。

在我心情不好没有空的时候，所有的问题都看不见就没法帮着解决问题了。


04

—


家长引导


关于家长引导请参考鸡娃系列家长篇(3.家长如何帮助孩子解决不会做的题目）


1. 解决问题的效率：高

7×24贴身服务。


2. 解决问题的能力：费家长模式，解决问题能力视家长能力。

大多数情况，随着年级越来越高，即使是强力辅助型家长解决问题的能力也未必能比孩子强多少。


3. 解决问题的质量：高

家长能清楚的了解孩子现在学习的情况、薄弱点，并做针对性处理。


4. 解决问题的耐心程度：家长想很耐心，但实际情况往往呵呵

处理不当易影响亲子关系。



05

—


使用搜题软件


关于搜题软件请参考关于刷题（3.用搜题软件解决问题）。


1. 解决问题的效率：很高。

方便、快速、提高孩子学习效率。

对孩子学习的即时反馈，避免减少孩子的学习热情。


2. 解决问题的能力：强

海量题库，此处略去广告词语。


3. 解决问题的质量：视问题和搜题软件情况。

有些问题有详尽的解答步骤和视频讲解，容易定位细节错误。

孩子很容易知道自己曾经在那些题目中碰到问题。

通过搜题软件的学习不系统。

有些问题只能搜到答案，且并不能确定是否正确。

存在部分题目解法繁琐、答案错误会给孩子带来误导。

自制力差的孩子容易抄答案，对搜题软件有依赖性，畏难。


4. 解决问题的耐心程度：搜题软件一定不会发脾气。



06

—


家教、陪练、托班


关于家教、托班请参考不同类型的补课方式比较。

关于陪练请参考孩子的学习，需要「女排精神」


1. 解决问题的效率：中等。

大多数情况线下1对1家教或者陪练并不是每天都有，解决问题会有延时。线上1对1家教或者陪练约课相对方便，解决问题效率较高。

大多数情况家庭请家教是有额外的学习任务，并不就是为了解决孩子平时的问题。

大多数情况托班只能解决校内的基础问题。


2. 解决问题的能力：视家教、陪练、托班的具体情况而定。


3. 解决问题的质量：视家教、陪练、托班的具体情况而定。

大多数情况，家教和陪练解决问题会更有针对性。

好的陪练会根据孩子的错题、难题、知识点误区以及练习中的坏习惯做针对性的处理。


4. 解决问题的耐心程度：很耐心

以前睿的语文学习辅导都是睿妈负责的，但她已经吐槽过N次“我也是理科生，为什么现在要辅导语文。”

第一次睿上刻意练的陪练课后，老师就指出了睿的一些学习习惯问题：

  • 阅读文章时应该把笔拿在手上，注意力更集中，圈划出重点线索词语，句子。

  • 平时练习不要涂画，也不要用修正带，养成用红笔/铅笔订正的习惯。

  • 审题时，也要在题干上圈划出关键词，标注关键信息，不然答题不完整。

在后续的陪练课程中，刻意练根据睿的学习情况，制定了循序渐进的练习计划，陪练老师对孩子小古文、阅读、作文的引导显然远远超过我们两个半瓶醋。

睿现在每周坚持上一次语文陪练课，在老师的陪伴下，提升了速度，提高了质量，增加了兴趣。我们一致认为以后语文作业的辅导就交给陪练能让家庭更和谐。


刻意练适合小学、初中孩子，双十一有活动（19.9元两节课），有兴趣的家长可扫码

注意事项


1. 请教应有礼貌，态度认真。

我见过不少家长请教群友时毫无缺乏礼貌，一个“请”字都舍不得说，好像是群友欠了他的钱现在在还债。


2. 做好准备工作，提高问问题的效率。

避免做伸手党。

比如孩子查一下教材和笔记就知道答案的问题。

比如家长在搜索引擎、搜题APP上查一下就知道答案的问题。


做好准备工作。

学会言简意赅的把问题表达清楚，有自己的思考和想法，减少占用他人的时间。

有必要的话需准备纸笔记录。


问问题的建议方式。

（老师）您好，打扰了，

我（孩子）遇见一个什么题（原题拍图，尽量不要手抄题，避免抄错），

我（孩子）已经解决了那几步，卡在什么地方。

（有会做这个类型题目的家长吗，）能抽空帮我解答下吗，谢谢！

如果原题有答案或者解答最好能一并贴图，减少其他人思考问题时走弯路的时间。

大多数情况，只求解题过程和答案或解决问题的相关资源即可。别硬撑着让群友从回答问题变成上课，毕竟谁都不愿意从盘古开天辟地开始补起。

对于某些群友的问题，我一点都不想回答，因为她们能问出来十万个问题，都舍不得活动一下自己的脑子。


3. 无论是否解决问题，表达谢意。

学习群里的家长没有义务回答其他人问题，在自己的问题解答之后，发红包才是表达诚挚感谢的正确方式。


4. 其他。

孩子请教老师时请注意场合。

看老师是否有其他事情忙，不要耽误老师时间。


孩子请教老师时不要有心里负担，害怕向老师问问题。


孩子请教老师时要有针对性。

  • 不建议向数学老师问非数学学科问题。

  • 尽量避免跨阶段问问题，除非老师是跨阶段老师。

    比如不建议向小学数学老师问初中、高中、大学数学问题，不建议向初中数学老师问小学、高中、大学数学问题等。

  • 尽量避免向校内老师问竞赛类问题，除非老师擅长此内容。

    比如不建议向小学校内数学老师问小奥问题。



08

—


总结


鸡娃不外乎以下三种模式

1. 费娃模式

  • 请教老师、同学是性价比很高的方案。有问题就应该大胆的问出来。

  • 不建议小学阶段的孩子使用搜题软件，初高中阶段的孩子使用搜题软件需要孩子自己更多的总结和思考。


2. 费家长模式

  • 家长可以通过学习群讨论问题和孩子一起成长，但对孩子来说并不是效率高的解决方案。

  • 家长引导是效果很好的方案，但对家长的要求很高。

  • 家长也可以通过搜题软件验证自己做的方法是否最优、答案是否正确、通过答案反推思路、检查孩子的作业是否正确。


3. 费钱模式

  • 如果既不想费家长，又想少费点娃，就要考虑费钱模式——家教、陪练等方案。
