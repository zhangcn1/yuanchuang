学前阶段的启蒙规划

学前阶段的启蒙之路往往需要家长制定出切实可行的规划路线，并按照这一路线坚定不移地执行下去。在本文中，孩子妈妈详实分享了自己的高效规划和孩子的巨大进步，尤其是在语文和英语方面实现了顺势而为的无痛启蒙，更是值得大家学习与借鉴。文末附群管建议，供各位参考。


「Major Snow」

{ 基本情况 }

孩子情况

女孩，上海地区，大班在读。性格温和，专注力好，能很好地完成学习任务，时间观念强。

 

家庭情况

妈妈：理工科，工作时间稳定。负责收集资料，制定各科规划并落实。

爸爸：理工科，经常加班出差，基本没有时间参与鸡娃。

 

目前进度

英语：自读RAZ L及高阶桥梁，亲子共读RAZ P，听力初章。

语文：已突破桥梁书阶段，可自读文字类书籍。

数学：启蒙二组在读。

 

小升初目标

一线理科班。


「Major Snow」

{ 启蒙回顾 }

英语启蒙

孩子2岁左右开始英语启蒙。

最开始选用《海尼曼》《牛津树》，每天半小时亲子阅读，几套书穿插进行。大概1年半读到RAZ J时，孩子有了自读的意愿，于是我便开始教她高频词和自然拼读。



有了前面1年多的指读铺垫和积累，孩子的高频词掌握得很快。自拼我们使用Alphablocks和《蒲公英拼读王》《牛津自然拼读世界》，通过“看视频-跟读教材-隔天复习”的形式，孩子在半年时间内便可熟练运用自拼规则。

自拼规则熟练后，我拿出之前读过的低阶分级和绘本，让孩子先自己拼读，再听音频校准读音，熟练朗读后进行录音。每套书大概自读2~3遍，G1阶段共用时1年左右完成。


英语启蒙的反思

从开始启蒙到完成G1阶段自读，孩子前后经历了3年的时间，过程很顺利。如果再给我一次机会重新启蒙，我会选择在以下几个方面进行优化：

  • 加强时效规划。听力词汇积累到1000词时就应该加入自拼，同时加入自读，提高效率。

  • 避免低阶反复。应该将亲子阅读的书单和孩子自读的书单进行区分，并且随着孩子能力的提高及时更新书目，不用读完再买。

  • 加大熏听时间。自拼开始后加大熏听力度有点晚了，早点开始能够帮助孩子更快地扫清阅读上的障碍。


语文启蒙

孩子1岁多开始语文启蒙，最开始是亲子阅读各种绘本。3岁时孩子迎来了识字敏感期，我便顺势使用《四五快读》带孩子进行系统认字，3个月左右学完六册，快速积累了大约600字。

认字书结束后，我们用《日有所诵幼儿版》《青蛙弗洛格》《不一样的卡梅拉》等简单有趣的图画书帮助孩子自读，同时选择《彩色世界童话》亲子共读，从开始一人读一段过渡到一人读一页，在阅读中继续积累认字。



语文启蒙的感悟

从识字到自读，其实是很长的一段路。用识字书积累的几百字不足以支撑自读，所以我用小学一二年级的语文课本帮助孩子铺垫之后，再去挑选了桥梁书让孩子自读。

在此期间我的感悟是，孩子自读的书不能跨度太大，亲子共读也需要多陪伴一段时间。


数学启蒙

中班之前，孩子的数学启蒙基本没有规划，只是学会了从1数到100。在升中班的暑假，我偶然结识了睿爸小屋，就此开始了孩子的数学启蒙。

彼时恰逢启蒙小组2组招生，为达到入组门槛，我立刻用《三招过关》《高思课本》等教材开始自鸡。4个月后，孩子完成了校内一二年级的计算部分和《高思课本》一年级的内容，勉强够到了小组的门槛，我便果断报名了。



「Major Snow」

{ 中班进展}

英语

今年初，孩子的英语阅读进入G2阶段。这个阶段以RAZ为主线，同时兼顾同级拓展，并会对一些较难的主题进行提前铺垫。

另外，由于我们使用的RAZ G2是按照蓝思值排列的合集，有些级别读到后面实在太难。因此从L开始，我会让孩子先读完每个级别的前4本，一直读到P时再回头阅读L级别的后续文章。这种螺旋式的上升方式很适合孩子不断进阶。



语文

中班将学习重点放在了英语和数学上，语文阅读只保留了自读。每天读一本桥梁书或一两个小故事，进展缓慢。

另外，中班下学期开始学习《小学生必背古诗词75+80首》。每天睡前15分钟，我会带孩子讲解一首新诗，同时复习前面已读的内容，8月份时已完成第一轮学习。


数学

由于提前铺垫了计算，刚开始的几周小组学习的难度并不算大。到了应用题部分，煎熬正式开始——孩子虽然可以自己读题，但是无法理解题意。

为了更好地提高效率，我借鉴群友分享的方法，将小组学习调整为“复习前一天的错题2-3道，再做新题6-7道”的模式。在开始做打卡卷前，我也会让孩子把错题再讲一遍思路。采用这种模式以后，孩子每次都能独立完成打卡卷，正确率80%左右。

为了让孩子学得更扎实一些，我们又跟着启蒙3组每周复习一次，主要做拓展卷及打卡卷。希望通过复习，把学过的知识点彻底搞懂学透。

{中班暑假总结 }

由于中英文阅读都遇到瓶颈，无法突破章节书，所以今年暑假孩子的学习重点就放在了中英文的阅读上。

假期时间规划

  • 白天

    • 中文阅读：1小时

    • 英文阅读：1小时

    • 数学计算：15分钟

  • 晚上

    • 英语录音：20分钟

    • 数学学习：1小时

    • 英语共读：1小时

    • 写      字：20分钟

    • 古诗古文：20分钟

假期学习情况

 英语

  1. 听力：主听初章，每天2小时，根据情况调换听单。

  2. 录音：每天录音20分钟，主要是亲子共读过的书。

  3. 共读：完成RAZ O1-O4、P1-P4。每天晚上读1-2篇，重点讲解科普类文章，虚构类自读居多，偶尔解释长难句。

  4. 自读：默读一两本高阶桥梁，如Monkey me、Owl Diaries、JBJ等。


 语文

  1. 自读：暑假读了《晴天有时下猪》《没头脑和不高兴》《利奥叔叔历险记》《胡椒罐大楼的小侦探》《最美中国童话》等，顺利实现了默读。

  2. 小古文：使用《小古文100篇》拓展材料及视频精讲课，按照“正音→看视频讲解→朗读注释和译文→朗读拓展小知识”的模式进行，并坚持每天睡前复习。 

  3. 写字：使用童嘉优课的视频进行学习，先看视频，观察字的结构，了解书写要点后描红5遍。每天坚持20分钟，暑假写完了一上的字。


 数学

暑假继续跟小组进度学习，主要完成了30-39周新练习卷和打卡卷，复习了1-12周内容。

在时间安排上，孩子周一和周二完成基础卷部分，具体操作为：看视频→自刷例题→听懂讲解→独立做基础卷→核对答案→搞懂错题。

周三周四孩子完成拓展题部分，其中部分难题需引导完成，并按照之前总结的方法循环复习。

周五做思考题并复习3组的拓展卷，周六复习3组的拓展卷和打卡卷，周日重做错题，打卡并听讲解，重做错题。


 文体

每周1次画画课，2次跆拳道。

每天坚持1小时户外。


假期总结

英文共读RAZ P＋每天自读一两本高阶桥梁，今年距离站稳初章应该不远。

语文实现了默读，简单的儿童文学也可以很顺畅地读下去了。

综上，暑假希望实现默读、突破章节书的目标已基本完成。

当下最大的困惑是数学规划的问题。

  • 现在孩子每周的数学学习时间为8-9小时，对于启蒙是否足够？

  • 孩子数学前学后忘，虽增加了复习，但毕竟时间有限，是继续往前学还是复习旧知识？

  • 由家长引导的学习，效果是否打折，是否需要定期测试学习效果？

另外，在英语方面我也有一些困惑。

  • 是应该选择继续稳扎稳打，还是每个级别挑一部分读即可？

  • G3篇目太长，如何提高效率？

  • 是否需要增加外教口语课，还是等中章后增加综合教材？


「Major Snow」

{ 心得感悟 }

回顾孩子的中英文启蒙，其实大多是顺势而为，抓住了孩子的几个敏感期。现在想来，在兼顾孩子能力和兴趣的情况下灵活做出书目的调整，是这一阶段做得比较成功的一点。孩子在读书上没有感觉到压力，兴趣一直很浓厚，进展也很顺利。

不过，在中班开始抓数学以后，孩子的中文阅读有了近一年的停滞。如果可以重新规划，中英文的前期阅读可以安排得更合理紧凑一些，也可以有一些时间做适当的数学启蒙。

对于数学，在启蒙小组近10个月的努力下，孩子数学进度赶上了不少，数学思维也有了很大的变化，从开始的畏难到现在可以独立做题，一直在坚持中前进。每天看看小组群里家长们的讨论，看看其它孩子的进展，我也汲取了很大的动力。

最后，祝小屋蒸蒸日上，孩子们都能梦想成真。


群管建议


孩子在学前阶段取得了不俗的成绩，这也与父母的规划和执行密不可分。现针对当前困惑回答如下，供参考。

  1. 数学方面，从孩子现在的年龄来看，当前的掌握程度已经非常不错。时间分配以预习理解新课为主，复习可以等到综合卷发现问题时再进行针对性训练。部分低年级知识点起到锻炼思维的效果即可，将来并不会考到。另外，建议积极参加机构考试，积累线下考试经验，顺便检查之前的知识漏洞，查漏补缺。

  2. 英语方面，RAZ可以继续坚持默读，从其中抽取部分内容进行朗读，朗读保证每天10-15分钟即可。同时增加桥梁书和章节书的默读量，尽快进入初章。可以考虑给孩子测试一下AR或蓝思值，根据测出的结果选择适合孩子阅读的书目。如果桥梁初章的默读量足够，RAZ可以只读NF类，G3后可尝试跳过亲子阅读，直接让孩子自行默读，对于quiz成绩不理想的，家长再参与共读。至于综合教材或精读的外教课，听读入中章后可以考虑加入，但要注意魔都XSC对口语和写作的需求不大，建议优先提升听读能力，后期再补充语法专项学习。
