群友：上海红豆妈
文章链接：学前原版娃的小学语文学习
发文时孩子年级：二年级
文章适合学段：学前，小低
文章适合学科：语文
关键字：语基，阅读，作文

文章部分要点总结：
1. 语基（读写背默）：
默写可以用《默写小达人》，妈妈牌默写纸可以搜“看拼音写汉字”查缺补漏。教辅一课一练和飞翔。
2. 阅读：
小低年级语文试卷做不好，解决方案不是去多做题，还是要从学习使用这门语言来着手解决，那首先是广泛的阅读，再往后需要主题广泛涉猎，同时注重知识性和文学性。
要带着“问题”或者说“思考”读书。读书后的总结（概括和理解）需要家长引导。
小低阶段主要任务是见识汉语的多样性和丰富性，经典的故事、寓言都可以读。
书单：阅读书上片段节选、新课标书单和平哥书单。
3. 作文：
有了日常口语小故事做基础，再学学看图写话的套路、语言表达的结构，基本上交出来的作文就达到优了。
3.1 看结构是否合理。学校会教总分总的结构，这个大框架跟着学校来。
3.2 看语句是否通顺。让孩子多读几遍，基本靠语感。
3.3 是否有错别字、病句；语言逻辑是否正确。比如“得、的、地”的使用，比如标点符号的使用。
基本的语句合规之后再看点题、承上启下、描写、感悟等的高阶一些的结构和语言表达。

群友：陕西嘻嘻妈
文章链接：焦虑妈妈的成长之路
发文时孩子年级：二年级
文章适合学段：学前，小低
文章适合学科：语文
关键字：写字，阅读，作文

文章部分要点总结：
1. 写字：
1.1 老师有耐心，家长更要有耐心。
1.2 不要去20、30多人的大班，一节课仅单独指导孩子两三个字的放弃。
1.3 距家不要太远。比起数学英语，练字是容易被放生的那个。
1.4 不妄想一个假期就能收获一个字迹整洁漂亮的孩子。练字整齐，写作业打回原样时，不要气馁，坚持个半年一年就成型。
2. 阅读：
阅读理解一年级就开始练，寒暑假调整级别高于一个年级的做。
3. 作文：
3.1 写话写作文首先不要想这太难，我无处下手。
写作文就是把日常的口语转换成文字，转换的时候删除重复的影响美感的话语。
文章一定要避免一堆错别字，看不清的标点符号。
3.2 注意划分段落。
不要一大段写4幅图或者两三件事，拿起来需要一口气读完的文章，第一印象会扣分。
一篇文章前后上下，都是相互关联串起来的。前面的内容和后面的收尾是遥相呼应的关系。
3.3 说完基本注意事项，再说具体操作。
写作文之前，先动脑、再动笔。先打个框架，落笔时跟着框架走。
框架分三部分：
首先是总介绍，引起下文。
中间是详细介绍事情的经过、过程。
最后的总结，谈谈感悟理想、自己将来要如何、表达自己的赞美之情和正向观点，是一篇文的灵魂。
写作文要制造悬念，特别是叙事类文章，起起伏伏引着读者走。
3.4 写作文不要刻意限定字数，练笔多写。
日常写话多写一些内容，后期再删减，等下次遇到同类的作文，直接修改就是一篇更好的作文。
3.5 收尾要收的漂亮。
切不可前面认认真真思考，结尾处随意两句打发读者。

群友：北京Y妈
文章链接：三年级普娃语文“出坑记”
发文时孩子年级：三年级
文章适合学段：小学
文章适合学科：语文
关键字：语基，阅读，作文，单元复习

文章部分要点总结：
规划：小低搞定英语，小高再追数学（校内优秀略提前，没有小升初择校压力），语文日积月累。
1. 夯实字词基础
1.1 寒暑假用教材同步字帖让娃自学下学期生字，并同时起到练字的功效。
1.2 用《默写能手》同步校内字词，每天1-2页。
1.3 用anki每天听写，根据记忆曲线定期复习字词。
关于自建词库和听写过程中需关注的点请参考原文。
2. 阅读：
保持高质量的输入（诵读），输出部分（阅读理解题）则紧跟校内老师。
2.1 现代文诵读
寒假每天诵读1-3篇现代文，内容为《黑马阅读》、《孟建平》、《一本》等阅读理解练习册。
家长：陪读，领读，各种彩虹屁。
2.2 文言文诵读
寒假每天诵读3-5篇文章，遍数不等，直至把它们读熟。
家长：每周挑一天，带着娃一起诵读、讲解文章意思、重点字词意思。
2.3 阅读理解
对于平时阅读量本就不算很大的小男孩而言，不要期望过高。
家长：题目的标准如果把握不好，可以向老师请教。老师的回复和讲解都非常细致，比参考答案更容易理解，家长看懂后会再解释给孩子。
3. 作文
3.1 初始阶段不讲求任何套路，动笔前会帮助孩子去梳理思路。
3.2 整理好思路后，动笔写时，如果孩子需要，家长会帮着打磨一下字句，加一些“好词好句”。
3.3 每次写完作文，鼓励小朋友自己声情并茂的朗读一遍。
4. 单元复习
使用亮点给力大试卷（基础卷）查漏补缺。
家长：不要在意成绩和排名，更关注孩子通过长期的积累和及时的复习，在考试中稳步提升，取得进步，增强学习兴趣，重拾自信！

对家长的建议：
1. 帮助孩子从字词句打好基础；
2. 提供高质量的泛读资料；
3. 陪伴孩子一起记录生活中的点滴趣事和真情实感；
4. 及时复习巩固争取校内成绩优良，进一步激发孩子的学习兴趣和自驱力。

群友：上海千妈
文章链接：古诗背诵从考级开始
发文时孩子年级：三年级
文章适合学段：小低
文章适合学科：语文
关键字：古诗文，背诵

文章部分要点总结：
1. 通过学习群打卡考试获得奖金增强孩子的学习积极性。
2. 从硬背到自学背诵。
3. 记忆曲线辅助。
4. 一遍画图一遍记古诗。
5. 让孩子复述一些诗句比较长的、有画面感的古诗，体会诗人的情绪和理解古诗的意图。
6. 参加活动的好处：自觉性有提高，学习态度很积极，自学能力变强了。

问题：孩子急于完成背诵，不看注释不学习古诗词义和译文，完全违背了当初学习古诗参加考级的初衷。
解决方案：从学习目的出发给孩子强调内容比形式重要，在背诵过程进行字词的抽答，在背诵完成后综合提问或者请她讲述对诗文的理解、对诗人的了解等等，包括前面提到的用“我是小讲师”方法复述记忆。

问题：遇到周末及国庆节假日，家里其他活动安排多了，反而容易把每日的背诗学习拖延疏忽了。
解决方案：屡次反复，实行得最好的情况还是每天能有固定的时间背诵比如晨读，形成学习和记忆的习惯。

群友：江苏Q妈（高中语文老师）
文章链接：迷雾中的语文学习
发文时孩子年级：三年级
文章适合学段：小低
文章适合学科：语文
关键字：古诗文，现代文，文化常识，作文

文章部分要点总结：
先分析孩子语文学习的问题：
1.课前预习过于简略。
2.听课低效。
3.课后巩固几乎没有。
4.作文缺失。

语文学习规划：
1. 古诗：在理解的情况下，再谈背诵。
推荐：《俞老师古诗词阶梯阅读训练》
2. 古文：
第一轮《小古文100篇》。
第二轮《小古文大过关》。
第三轮采用上海群友推荐的《6-7年级满分突破》。
关于古文背诵，我是极其反对小学背初高中长篇的古文，还有小学生学习《古文观止》等难度比较大的文章。
具体操作和原因请参考原文迷雾中的语文学习。
3. 现代文阅读：
3.1 信息化阅读：
实用类文本：平时孩子读完文章或者一些新闻报道后，能够精准复述内容。
论述类文本：和孩子讨论有哲理的漫画、名言，培养思辨能力。
3.2 文学类阅读：
学前主要读语言比较优美的绘本。
大班读一年级的必读书目。
一年级以国际获奖小说为抓手，孩子喜欢的反复多次看。
二年级开始，有意识选择名家的儿童作品作为主线。
帮孩子选书，个人推荐从年度好书榜单中按需索取。初高中的孩子可以从年度散文，年度小说中选择自己喜爱的作品。
4. 文化常识：
推荐：《日积月累》，《中华优秀传统传统文化经典读本》，王力的《中国古代文化常识》(主要家长读)。
5. 写作与升格：
一篇好的作文首先要文从字顺，其次是详略得当，再次是选材精彩，最后是独特的写作风格(对小学生来说有点拔高，可以放在初中再践行)。有条件可以看一些的升格作文的范例，找到升格前后的差异，从而可以有效指导。
从应试角度推荐：《作文与考试》

群友：江苏Daisy妈（小学语文老师）
文章链接：四年级娃中文阅读之路
发文时孩子年级：四年级
文章适合学段：小学
文章适合学科：语文
关键字：阅读，历史，科普，文学

文章部分要点总结：
1. 历史系列：先整体后细节。
文史不分家。语文是平面的历史，历史是纵深的语文。孩子对世界的认识和了解，离不开历史的根基，文学作品的阅读理解也离不开历史背景知识的铺垫。
低年级的时候，激发兴趣，迅速向文字书进阶。边读边梳理出中国史的整体框架，画出大树的主干，然后在阅读中不断填充细节。中高年级再向世界史拓展。
2. 科普阅读：兼顾经典和新知。
小学阶段，尤其是低年级，鼓励孩子尽可能拓宽阅读面，注意知识结构的平衡，思想和知识体系的搭配。为什么强调低年级呢？因为低年级的孩子，基本上家长和老师给什么就读什么，都能读得津津有味。随着孩子认知的发展，四年级上学期开始，阅读个性越发凸显。四年级之前，父母老师大有可为。一般的孩子都能够接受有情节的故事阅读，对于科普阅读，我们也不可偏废。
科普阅读与考试，与后续的学习进阶密切相关。大千世界万物皆有奥秘，基础的科普认知可以帮助孩子认识客观世界。
3. 文学类阅读：筑桥搭梯再拓展。
4. 其他：
作为老师，要对孩子的某一阶段、一学期的学科课堂学习情况负责。
作为家长，陪伴孩子一路走来，整体、贯通的思想是非常重要的。
四年级之前，孩子学习阅读，四年级之后，孩子在阅读中学习。
阅读，不仅仅是指读纸质书，还可以听音频。
一切的推书，不要强求。阅读兴趣，要以允许自由选择为前提。
阅读的时机很重要。
图书的版本很重要。
