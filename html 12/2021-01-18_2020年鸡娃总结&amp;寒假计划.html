2020年鸡娃总结&寒假计划

原创睿爸小屋睿爸小屋2021-01-18 07:59:00

编者荐语：帝都顶妈鸡一年级男娃的年终总结&寒假计划。想抄作业的学前小白家长们，看看帝都鸡娃的常态颤抖吧！


从2020年开年就过的鸡飞狗跳，从没太平过，眼见娃今年入学，身处帝都大朝阳的我无比焦虑，从娃上幼儿园起就打算冲ZZP（早早培），今年上半年是最后的冲刺了，眼见身边的牛娃英语考过KP（KET&PET），数学至少跨一级创A，我家的小渣渣还在为疫情期间不能出去吃喝玩乐在发脾气。如鸡娃前辈所说，假期是检验鸡娃效果最好的时机，我自然不能放过。


【幼儿园大班寒假】


数学：

刚退掉了线下跨级的创新班，不是娃的问题，是我身体的原因没办法继续坚持鸡娃了。刚退班时，娃不用刷题、没有了课前测、没有了作业、没有了老母的狮吼，他开心的不得了。但没过两周，娃开始想念老师，想念同学，但为时已晚，名师的课退了就别想再进去了，退而求其次，报上了同一名师春季的线上课，算是对娃的弥补。

1. 自学完成了一年级校内，背熟表内乘法。

2. 刷了大半本一年级奥精。

3. 每天至少一篇一创难度的口算（100道题）。

4. 娃一直蛮喜欢火花思维，但现在对娃来说过于简单，咬牙上完。


语文：

语文一直是娃的弱项，就靠着幼儿时的广泛绘本阅读在支撑，不爱自主阅读，不要背诗，不写字，不听故事，唯一的优势就是学会了拼音。

1. 每天至少一篇一年级日有所诵指读。

2. 背诵古诗10首。


英语：

1. 机构课：完成了自拼中级线下课程、本想连上自拼高级但以为疫情原因泡汤了，初级是在春节前1月份上完的，那时候疫情还没有被爆出。

2. 完成KET单词默写。

3. 完成180天阅读1级、2级练习册。

4. 阅读：完成RAZ-G，树屋前10本，365小故事只读了前三本（娃觉得简单就放弃了）。

5. 听力训练：每天至少1小时。

【幼儿园大班春季开学后】


由于是幼儿园大班下半学期，没有开园，娃一直在家，我上班后，白天我留作业由姥姥陪学，数学部分增加了拾伍课堂豆包数学课程共14讲，继续优才一创的学习；语文增加了写字练习，但效果甚微；英语则按部就班。


数学：

1. 每周完成优才一创线上课程，包括课前测和课后作业。

2. 从3月份起，每周两节拾伍课堂豆包课程。

3. 每天一篇一创口算。

4. 自学完成校内二年级上册。


语文：

1. 每天逼着完成朗读：日有所诵、小古文100篇。

2. 看娃心情好时进行写字练习。

3. 背诵古诗20首。（由此能看出来，娃是真心不喜欢语文）。


英语：

1. 机构剑一线上课，巩固KET单词。

2. 阅读：RAZ-H、I（未完），神奇校车2、小猪梅西、奇先生、狗人、科一。

3. 听力：每天至少1小时。

4. 三月份AR测试3.3。

【升一年级之前的暑假】


ZZP考试尘埃落定，意料之中简历关都没通过，自此妈妈佛系起来，不再逼娃奥数，这一来感觉时间宽裕多了，也为即将入小学开始了各项准备。


数学：

参加了各机构的测评，最后选择了优才一年级创新A+班和拾伍课堂的全年豆包班（线上）。

1. 完成机构集训优才8次课，豆包12次课。

2. 每天完成二年级口算一篇。

3. 自学完成校内二年级下册。

4. 继续刷一年级奥精。


语文：

1. 进行拼音系统复习。

2. 引导自主阅读，使用分级读物《我爱阅读》（蓝色）。

3. 凭娃的兴趣穿插进行：写字、古诗背诵、小古文、黑马阅读、句子训练。


英语：

1. 线上剑一复习集训、自拼高级学习。

2. 阅读：RAZ-I、J（未完），第三图书馆，黑猫分级阅读初级。

3. 词汇：4000第一册前五单元单词。

4. 听力：每天至少1小时。

5. 7月份AR测试3.0。


【一年级入学后的秋季】


家门口渣小，重点把心思用在了校外。


数学：

校内的数学基本没有辅导，分配给数学的时间全部用来完成辅导班的作业和刷题了。

1. 每周末优才线下课，包括课前测，完成作业、附加讲义。

2. 每周两次拾伍课堂豆包线上课，包括课前测、作业；练习册没时间做，十月份后因课程冲突没有按时上课，计划寒假突击补课+完成练习册。

3. 每天一篇口算，目前难度为多位数乘除1位数。

4. 一年级小白本完成了10讲，二、三年级举一反三1000题各完成三个相同模块练习。


语文：

上学后发现语文是个大窟窿，听说读写，样样补起；校内开始几乎没有作业，从12月份开始留笔头作业生字的笔顺练习，1月11日期末考试是乐考。

1. 课内生字会组词，课文会背诵，古诗能背70多首了。

2. 教辅买了默写小达人和期末冲刺100分，几乎没用上（计划寒假刷完）；句子训练做了一上的一部分；阅读训练用的黑马阅读和北大阅读课，对娃来说有些偏难，基本用于读文章了，没起到作用。

3. 阅读：《我爱阅读》系列蓝色，《汉声中国童话》1、2月，《这就是物理》。

4. 文言文方面：小古文100篇没怎么推动，《给孩子讲论语》第一册上篇《学而第一》。


英语：

重心放在了校外培训班，校内英语只完成APP作业。

1. 每周剑二线下课，Guess What3，作业：练习册、单词短语会默写，每周一次口语练习。

2. 口语：目前线上外教每周一次，内容：科一。

3. 阅读：RAZ- J、K，第三图书馆（完成），继续树屋，A to Z和I Survived各看了前三本。

4. 9月份AR测试4.4，12月份3.3；咨询了娃的英语老师，波动大的原因很可能跟上学后没有大量时间进行阅读有关，这部分是寒假的重点。


【一年级总结&寒假计划】


娃马上迎来了一年级的第一个大考，虽然是乐考，但他对学习、对考试有了新的理解，在思想上也重视起来了，把语文书上的课文全部背了下来（可能是在校外学了数学和英语的关系，我跟娃只复习了语文学科）。通过一个学期的学习，娃的学习习惯还需要改进，最好不要遇到不会的题就喊妈，就不往下做了，希望娃能学会主动思考、学会动笔、学会试错。


数学：

计数是薄弱，数图形总会漏数，枚举不愿意动笔，需要加强练习。寒假除了完成培训班集训课程和作业外，另针对薄弱板块进行刷题练习，小白本、举一反三1000题重做一遍；完成校内三年级课程。


语文：

用钱儿学古诗音频成功勾起了背诗乐趣，寒假趁热打铁，争取背100首诗；完成一下一、二类字组词、默写，完成一上句子训练；阅读完成《我爱阅读》系列黄色、《这就是化学》，根据娃的吸收情况继续推进《给孩子讲论语》。


英语：

自学新概念一前60课，练习课文听抄；阅读RAZ-L、M，完成树屋前28本；保证每天听音频至少1小时，每周至少三节口语课。


体育：

因疫情的关系不去游泳和打篮球了，户外晒眼睛吧。
